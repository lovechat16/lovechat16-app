# プロジェクト詳細：ラブチャット16診断

## 何を作っているか

恋愛における LINE / DM の送り方・受け取り方のクセを16タイプに分類する診断ツール。
MBTIのように「タイプ名＋説明」の形式で、自分の恋愛チャットスタイルがわかる。

## 現在の実装状況（2026-06-08更新）

| ファイル | 内容 |
|---|---|
| `index.html` | メイン診断（自分のタイプ診断）+ chars-desc.js 組み込み済み |
| `partner.html` | 相手のタイプ診断 + chars-desc.js 組み込み済み（相手の会話像・あるある表示）|
| `aishou.html` | 2タイプの相性診断 + partner.html リンク追加済み |
| `kanri.html` | 管理ツール（恋愛事故調査ファイル下書き生成） |
| `chars.js` | 16タイプのキャラクター定義データ（共有データ・変更禁止） |
| `chars-desc.js` | 16タイプ結果文章（新規追加）intro / traits / strength / caution |
