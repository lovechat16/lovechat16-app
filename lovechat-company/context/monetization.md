# 収益化戦略

## フェーズ設計

### Phase 1：無料で拡散させる（現在〜）
- 診断ツール自体は完全無料
- 結果シェアで自然拡散
- SNSフォロワー・LINE登録者を積み上げる

### Phase 2：コンテンツ収益（拡散が安定したら）
- **note**：有料記事（タイプ別返信ガイド、相性深掘りなど）
- **タイプ別コンテンツ**：「○○タイプの人への上手な返し方」

### Phase 3：LINE / サービス収益（中長期）
- **LINE公式アカウント**：診断結果に応じた自動配信
- **有料相談・添削**：LINEの文面を見てアドバイス
- **サブスク or スポット**：「返信設計ガイド」定期配信

---

## 収益化の導線設計

```
診断ページ
    ↓
結果ページ（無料・シェア促進）
    ↓
「もっと詳しく知りたい？」→ note へ
    ↓
「返信の悩みを相談したい？」→ LINE登録へ
    ↓
LINE内で関係構築 → 有料コンテンツ案内
```
