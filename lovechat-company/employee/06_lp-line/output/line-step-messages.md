## 使い方

## Step 1：即時（登録直後・数秒以内）

```
こんにちは！ラブチャット16診断へようこそ🔥

LINEの返信スタイルで、あなたの恋愛タイプを16タイプに診断しました。

今後、このLINEでは…
✅ タイプ別の恋愛ヒント
✅ 相手の返信を読み解くポイント
✅ 「これを送ると引かれる」実録NG集

…を週1でお届けします。

まだ診断していない方はこちら👇
→ [診断URL]

まずは自分のタイプを把握しておいてください🙌
```

## Step 2：3日後

```
こんにちは！ラブチャット16です🌙

気になる相手のLINE、読めていますか？

「返信が遅い」「既読スルーされた」
でも…タイプを知ると読み方が変わります。

相手の返信スタイルを7つの質問で分析↓
→ [相手診断URL（partner.html）]

相手が「受け身タイプ（R系）」だと確認できたら
「こちらから動くだけ」で流れが変わりやすいです🔥
```

---

## Step 3：7日後

```
【相性について】

自分のタイプ × 相手のタイプで
「噛み合う点」と「すれ違いやすい点」がわかります。

たとえば：
✅ 積極型同士 → 盛り上がりやすいが、どちらかが引いてバランスを取る必要
✅ 積極型 × 受け身型 → 積極型がリードすれば成立しやすい
✅ 受け身型同士 → 会話が止まりやすい。でも居心地はいい

2人の相性を診断↓
→ [相性診断URL（aishou.html）]

「スコアが低くても大丈夫」です。
相性の数字より、すれ違いの原因を知ることの方が大事。
```

## 週次配信テンプレート（毎週火曜）

### バリエーションA：タイプ別ヒント

```
【今週のラブチャットヒント】

今週は「○○タイプ」について。

このタイプに送ると好印象な一言：
「[具体的な一言例]」

逆に送ると距離が開きやすい一言：
「[NG例]」

あなたのタイプは？ → [診断URL]

#ラブチャット診断
```

### バリエーションB：あるある系

```
【LINEあるある】

「既読ついてから3分でそわそわし始める」人、
これが「直感アタッカー型」の代表的なクセです。

あなたにも当てはまる？
→ [診断URL] で16タイプ確認

#ラブチャット診断 #恋愛あるある
```

### バリエーションC：Q&A形式

```
今週届いた質問（架空）：

Q. 相手がAFED（直感アタッカー型）なんですが、
   返信が遅い私でも大丈夫ですか？

A. 大丈夫です。
アタッカー型が本当に好きな相手には、
「来た時の温度」で判断します。
返信が遅くても「来た時にちゃんと受け止める」だけで、
関係は続きます。

→ 相手のタイプを診断 [partner.html]

#ラブチャット診断
```

## 配信頻度と注意点

- ステップ配信: 登録〜14日でほぼ完結（押し付けすぎない）
- 週次配信: 週1回（火曜19時）を推奨
- 配信頻度: 週2回以上はブロック率が上がるため避ける
- 「友だち限定」感を出す: 「ここだけの話」「LINE登録者先行」の言葉を使う
