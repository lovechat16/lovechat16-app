# ラブチャット16｜イントロページ CRO分析（CTR改善提案）
作成: 06_lp-line-designer
日付: 2026-06-08
目的: 診断開始ボタンの押下率（CTR）最大化

---

## 現状分析

### 現在のイントロページ構成

```
[バッジ] LOVECHAT 16 | 恋愛メッセージ診断
[H1] あなたの恋愛メッセージは、どのタイプ？
[リード文] 恋愛のメッセージ/DMには、続く人・失速する人の「会話のクセ」がある。
         20の質問で、あなたの会話タイプ・相性・事故りやすいポイントと、相性キャラがわかる。（長い）
[キャラ] 8体のキャラクター表示
[チップ] ⏱ 約90秒 | 📝 全20問 | 💞 16キャラ判定
[CTA] 診断をはじめる（ボタン）
[免責] ※この診断は人格を断定するものではなく...（控えめに ← Visual Critic指示済み）
```

### CTRを下げている要因（推定）

| 要因 | 影響度 | 根拠 |
|------|--------|------|
| リード文が長い（74文字） | 高 | スマホで3行になり、ボタンまでスクロールが必要になる場合がある |
| CTA文言「診断をはじめる」が平凡 | 中 | 機能説明にとどまる。ベネフィットが伝わらない |
| 社会的証明なし | 中 | 「〇〇人が診断済み」がない。信頼感に欠ける |
| チップの順番がユーザー心理と逆 | 低 | 「何が得られるか」→「手間は?」の順が自然 |

---

## 改善提案（優先度順）

### 改善1【最重要】リード文を短縮してボタンを早く見せる

**現在（74文字・3行）:**
```
恋愛のメッセージ/DMには、続く人・失速する人の「会話のクセ」がある。
20の質問で、あなたの会話タイプ・相性・事故りやすいポイントと、相性キャラがわかる。
```

**改善案A（32文字・1行）:**
```
あなたの恋愛メッセージのクセを、20問で診断。
```

**改善案B（39文字・煽り型）:**
```
既読スルーされがち？　返信が重いと思われてる？
その原因、メッセージのクセにあるかも。
```

**改善案C（刺さる系・26文字）:**
```
あなたの恋愛LINE、当ててみます。
```

→ 推奨: **改善案C**（最短・自信のある言葉・好奇心喚起）

---

### 改善2【重要】CTA文言の強化

**現在:** `診断をはじめる`

**改善案（比較）:**

| 案 | 文言 | 特徴 |
|----|------|------|
| A | `あなたのタイプを見る →` | ベネフィット明示 |
| B | `90秒で診断スタート` | 時間コストを前面に |
| C | `無料で診断する` | 無料強調 |
| D | `自分のタイプを知る` | 自己発見欲を刺激 |

→ 推奨: **案A「あなたのタイプを見る →」**（得られるものが明確で矢印で推進力あり）

---

### 改善3【中】チップの順番変更

**現在:** ⏱ 約90秒 → 📝 全20問 → 💞 16キャラ判定

**改善案:** 💞 16キャラ判定 → ⏱ 約90秒 → 📝 全20問

→ 「どんな結果が得られるか」を先に見せてから「手間」を提示する

---

### 改善4【参考】将来的な追加要素

ユーザーが増えてきたら追加を検討:
- 「累計○○人が診断済み！」（社会的証明）
- 「🔥 今日の人気タイプ：〇〇型」（話題性・ランキング感）
- 診断前に1問だけ「クイック質問」を見せて興味を引く
