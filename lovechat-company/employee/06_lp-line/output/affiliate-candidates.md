## 設定手順

### 1. ASPに登録

推奨: **A8.net**（国内最大のASP、マッチングアプリ系が充実）
URL: https://www.a8.net/

### 2. 提携申請

A8.net で「Pairs」「ゼクシィ縁結び」などを検索 → 提携申請。
審査は数日かかることがある。

### 3. chars.js の AFFILIATES を更新

提携承認後、発行されたアフィリエイトURLを取得して：

```javascript
const AFFILIATES = [
  {
    name: "Pairs",
    catch: "まじめな出会いを増やすなら",
    desc: "累計会員数No.1のマッチングアプリ。真剣な出会いを。",
    url: "https://ck.jp.ap.valuecommerce.com/servlet/referral?sid=xxxxx&pid=xxxxx",  // ← ここに実際のURL
    cta: "無料で始める"
  }
];
```

URLが有効な `http` or `https` で始まっていれば adCard が自動表示される。
