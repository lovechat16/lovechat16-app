# A8.net アフィリエイト 登録・設定手順書
作成日: 2026-06-08
担当: 06_lp-line
目的: A8.netに登録してPairs等のアフィリエイトリンクを取得・設定する

---

## 概要

1. A8.netに登録（ASP = アフィリエイトサービスプロバイダー）
2. マッチングアプリ（Pairs等）に提携申請
3. 承認後にアフィリエイトURLを取得
4. chars.js の AFFILIATES にURLを設定
5. 結果ページにアフィリエイト広告が自動表示される

---

## Step 1：A8.netの登録

1. https://www.a8.net/ にアクセス
2. 「会員登録（無料）」をクリック
3. 必要情報を入力:
   - メールアドレス
   - 住所（報酬振込のため本名・実住所が必要）
   - 銀行口座情報（収益が発生した場合の振込先）
4. メール認証を完了

**登録後にやること**:
- 「サイト情報」の登録が必要
  - サイト名: 「ラブチャット16診断」
  - サイトURL: 公開済みの診断ページURL
  - カテゴリ: 「恋愛・婚活」または「ライフスタイル」
  - サイト説明: 「恋愛LINEの会話タイプを16種類に診断するWebアプリ」

**注意**: サイトが未公開の場合、審査が通らないことがある。GitHub Pagesで公開後に登録すると安心。

---

## Step 2：Pairsへの提携申請

1. A8.net管理画面の「プログラム」→「プログラム一覧」
2. 「Pairs」で検索
3. 「提携申請する」をクリック
4. 申請理由を入力（例）:
   ```
   恋愛LINEのコミュニケーションタイプを診断するWebサービスを運営しています。
   診断後に恋愛に興味のあるユーザーが集まるため、マッチングアプリのご紹介に
   適していると考え、提携申請いたしました。
   ```
5. 審査結果を待つ（数日〜1週間）

**承認後にやること**:
- 「プログラム詳細」→「広告素材」から「テキストリンク」を選択
- アフィリエイトリンクURL（`https://ck.jp.ap.valuecommerce.com/...` 等）をコピー

---

## Step 3：chars.js の AFFILIATES を更新

アフィリエイトURL取得後、chars.js（変更禁止ファイル）の AFFILIATES 部分を更新。

**⚠️ chars.js は変更禁止リストに含まれる**
→ 07_dev-engineerに以下の内容を伝えて実装を依頼する:

```javascript
// 現在の設定（変更前）
const AFFILIATES = [
  {
    name: "Pairs",
    catch: "まじめな出会いを増やすなら",
    desc: "累計会員数No.1のマッチングアプリ。真剣な出会いを探す人に。",
    url: "https://例-アフィリエイトURLをここに",  // ← このURLを更新
    cta: "無料で始める"
  }
];
```

```javascript
// 変更後（実際のアフィリエイトURLを入れる）
const AFFILIATES = [
  {
    name: "Pairs",
    catch: "まじめな出会いを増やすなら",
    desc: "累計会員数No.1のマッチングアプリ。真剣な出会いを探す人に。",
    url: "https://ck.jp.ap.valuecommerce.com/servlet/referral?sid=XXXXX&pid=XXXXX",  // ← 実際のURL
    cta: "無料で始める"
  }
];
```

**URLが `https://` で始まっていれば adCard() が自動表示される。**
URLが `https://例-...` のままだと表示されない（意図的な設計）。

---

## Step 4：表示確認

1. chars.js更新後、診断を完了して結果ページを表示
2. ページの一番下にアフィリエイトカードが表示されるか確認
3. リンクをクリックしてPairsページに正しく遷移するか確認

---

## 複数アフィリエイトの設定（上級）

AFFILIATES 配列に複数追加することで、複数のアフィリエイトを順番に表示できる（確認が必要）:

```javascript
const AFFILIATES = [
  {
    name: "Pairs",
    catch: "まじめな出会いを増やすなら",
    desc: "累計会員数No.1。安心・安全な出会いを。",
    url: "https://[PairsのアフィリURL]",
    cta: "無料で始める"
  },
  {
    name: "ゼクシィ縁結び",
    catch: "結婚も視野に入れてるなら",
    desc: "リクルート公式。真剣な出会いを。",
    url: "https://[ゼクシィのアフィリURL]",
    cta: "無料で試す"
  }
];
```
