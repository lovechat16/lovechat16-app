# 06 LP・LINE導線担当（LP & LINE Funnel）

## 役割

診断結果からnote・LINE登録への導線を設計する。ランディングページとLINE配信を担当。

## 担当範囲

- 結果ページへの「次のアクション」設計
- LINE公式アカウントの配信コンテンツ設計
- ランディングページ（LP）のコピー・構成
- 登録特典・リードマグネットの設計

## 必ず読むこと

- `../../context/monetization.md`
- `../../context/target.md`
- `../../context/tone.md`

## 導線の基本設計

```
診断完了
    ↓
結果ページ
    ├── [シェアボタン] → X / Instagram
    ├── [相性診断へ] → aishou.html
    ├── [もっと詳しく] → note記事
    └── [LINEで受け取る] → LINE登録
```

## 結果ページに追加するCTA（コール・トゥ・アクション）

### CTA設計の原則
- CTAは最大3つ（多すぎると何もしない）
- 最優先：SNSシェア（拡散）
- 次点：LINE登録（資産化）
- 最後：note誘導（収益化）

### CTAコピー案

**シェアボタン**
```
「これ私すぎる」→ Xでシェア
友達のタイプも知りたい → LINEで送る
```

**LINE登録**
```
LINEで詳しい結果を受け取る（無料）
→ タイプ別の返信アドバイスをお届け
```

**note誘導**
```
あなたのタイプへの上手な返し方を知りたい？
→ タイプ別返信ガイドを読む（note）
```

## LINE公式アカウント配信設計

### 登録直後（Step配信）

| タイミング | 内容 |
|---|---|
| 即時 | 登録お礼 + タイプ別一言アドバイス |
| 3日後 | 「相手のタイプも診断してみて」 |
| 7日後 | 相性診断の解説 |
| 14日後 | note案内（初回割引） |

### 定期配信（週1）
- タイプ別の「今週の恋愛ヒント」
- 新しいnote・コンテンツのお知らせ
- 「あなたのタイプにはこんな人が合う」
