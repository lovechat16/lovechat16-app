# 第1章：相手のタイプ別「返信の型」

> 「型を知る」とは、マニュアル通りに動くことではない。
> 相手が気持ちよく話せる「チャンネルに合わせる」こと。

---

## AFED（直感アタッカー型）の相手に効く返信

### このタイプの読み方

感情を即座に言葉にして送ってくる。返信速度より「熱量」で話す人。既読後の返信が遅いとそわそわし、「怒ってる？」と確認してくることも。

### 効く返信の型

**「感情 → 質問」の2ステップ**

```
相手の感情を一度受け止める → その感情をさらに引き出す質問を返す
```

**具体例**:
```
相手: 「昨日あったこと話していい？めちゃくちゃ嬉しかった」
型:  「聞く！なにがあったの？！」
```

相手が「聞いてもらえた」と感じる前半 + 続きを引き出す後半。両方あると効く。

### この型が効く理由

アタッカー型は「感情の共鳴」を確認している。受け止めてもらえることで、もっと話したくなる。「？」で返すことで「この人は深く聞いてくれる人」になれる。

---

## AFEP（ノリよく沼らせ型）の相手に効く返信

### このタイプの読み方

テンションと話題の転換が命。「てかさ〜！」で全然違う話に飛ぶのが普通。盛り上がった時の熱量は本物だが、深い話は苦手。

### 効く返信の型

**「乗る → 広げる → 別の話に飛ぶ」のエンドレスリレー**

```
相手の話題に乗る → 少しだけ広げる → 「てかそれで思い出したんだけど」で飛ぶ
```

**具体例**:
```
相手: 「昨日の映画、笑いすぎて死ぬかと思ったwww」
型:  「え何それ気になる！どのシーン！？てかどこの映画館行ったの」
```

2つ以上の「続きたい」を仕込むと、会話が止まらなくなる。

### この型が効く理由

このタイプは「話が弾む人」を好む。弾ませる才能を見せる → 「一緒にいると楽しい」が蓄積される。

---

## AFCD（静かな直球型）の相手に効く返信

### このタイプの読み方

自分から行動するが、感情表現は少なめ。絵文字より言葉を選ぶ。「会いたい」をさらっと送れる数少ないタイプ。

### 効く返信の型

**「中身で誠実に + 自分の意見を1つ入れる」**

```
相手が言ったことを受け止める → 自分の意見・感想を短く添える
```

**具体例**:
```
相手: 「そうだったんだ」
型:  「うん。正直に話してくれてよかった。私も同じ感覚があった」
```

自分の考えを率直に入れることで「対等な会話」になる。

### この型が効く理由

直球型は「中身で話す人」が好き。絵文字や飾りよりも、言葉の中身で評価する。

---

## AFCP（テンポ職人型）の相手に効く返信

### このタイプの読み方

会話のリズムを作ることに無意識に長けている。既読後1分以内に短く返す。話題が途切れると自然に次をつなぐ。

### 効く返信の型

**「短く・速く・次の話題への橋渡し」**

```
短く返す → 「ところで」「そういえば」で次の話題を自然につなぐ
```

**具体例**:
```
相手: 「眠くなった？」（空気を読んで確認してきた）
型:  「ちょっとね！でも話したい。さっきの〇〇の話、もう少し聞かせて」
```

「眠くない + 話したい + 次の話題」を1文に圧縮できると完璧。

### この型が効く理由

テンポを命と思っている相手には、同じテンポで返すことが最高の「合ってる感」になる。

---

## ASED（じっくり情熱型）の相手に効く返信

### このタイプの読み方

返信は遅い。でも来た時の熱量が違う。下書きを3回書き直すこともある。「ちゃんと返したい」が行動原理。

### 効く返信の型

**「1つ以上、具体的に拾う + 自分の話も1つ添える」**

```
相手が送ってきた長文の中から「1つ具体的に反応する」
```

**具体例**:
```
相手: [2段落の近況報告]
型:  「〇〇の部分、ちゃんと読んだよ。最初はなんか複雑だったんだろうなと思って。
      私も似たような経験あってさ、[1文だけ自分の話]」
```

「ちゃんと読んだ」を証明することが最大のリターン。

### この型が効く理由

時間をかけて送ってきた文章を軽く流されることが、このタイプには一番「届かなかった」に感じる。逆に拾ってもらえると「この人に話してよかった」になる。

---

## ASEP（マイペース甘え型）の相手に効く返信

### このタイプの読み方

気が向いた時に全力で甘えてくる。返信ムラがあるが悪意はない。「ねえ聞いてー！」から会話が始まることが多い。

### 効く返信の型

**「受け止める → 軽く返す → 逆質問しない」**

```
甘えてきた内容を軽く受け止める → 重い逆質問はしない → 話の続きを促す
```

**具体例**:
```
相手: 「ねえ聞いてー！なんかモヤモヤすることあって」
型:  「聞く！なになに」（重くしない）
```

「どうしたの？大丈夫？」と心配モードに入りすぎると、相手が「大げさにしてしまった」と感じて引く。

### この型が効く理由

甘えを軽く受け取ることが「居心地のいい人」への近道。

---

## ASCD（熟考カウンセラー型）の相手に効く返信

### このタイプの読み方

相手の話を全て受け止めてから言葉を選ぶ。「それって、どんな気持ちだった？」が口癖。自分のことはあまり話さない。

### 効く返信の型

**「受け取った + 自己開示1つ + 相手の話に戻る」の三段構造**

```
相手の話を受け止める → 自分の体験・感情を少し話す → 相手に戻す
```

**具体例**:
```
相手: 「それって大変だったね。〇〇ってどんな気持ちだった？」
型:  「正直、最初はショックだった。あなたも似たような経験あったりする？」
```

相手に返す質問を混ぜることで「一方通行じゃない」になる。

### この型が効く理由

カウンセラー型は「自分が聞いてもらえること」を密かに求めている。開いてくれる人を貴重に思う。

---

## ASCP（余白キープ型）の相手に効く返信

### このタイプの読み方

詰めない・追わせる・余白を作る。追いかけるより追わせる側でいたい。大人っぽいが、実は少し不安なこともある。

### 効く返信の型

**「程よく短く + 謎を残す + こちらからは追わない」**

```
短く、少し意味深な一言 → 深追いしない → 次は相手から来るのを待つ
```

**具体例**:
```
相手: （たまに意味深な一言を送ってくる）「最近ちょっと変かも」
型:  「どのくらいの変さ？（笑）」（軽く返して、続きは任せる）
```

こちらが同じくらい「余白を持った人」に見えると、相手が引き寄せられてくる。

### この型が効く理由

追われることに慣れているタイプ。追わない人が珍しくて刺さる。

---

## RFED（反応濃いめ共感型）の相手に効く返信

### このタイプの読み方

来たら全力で共感する。自分からは送りにくい。返信が止まると「なにかしたかな…」とそわそわする。

### 効く返信の型

**「こちらから話題を振る + 全力の共感を引き出す質問」**

```
自分から話題を提供する → 「これどう思う？」「あなたはある？」と共感できる質問を添える
```

**具体例**:
```
あなた: 「最近こんなことあったんだけどさ。〇〇って経験ある？」
相手: 「あるある！！それめちゃくちゃわかる！！！」
```

待っているだけでは会話が始まらない。こちらから「始点」を作ることが鍵。

### この型が効く理由

全力の共感は受け身タイプの「得意技」。その得意技を使える場をこちらが作ってあげる。

---

## RFEP（わちゃわちゃリアクション型）の相手に効く返信

### このタイプの読み方

来たら全力で乗る。盛り上げ役が自分の使命だと感じている。自分からはなかなか送れない。

### 効く返信の型

**「面白い話題・ボケを振る → 全力で乗ってもらう」の繰り返し**

```
笑える話 / 共感できるエピソード を提供する → 相手が乗ってきた全力に乗り返す
```

**具体例**:
```
あなた: 「今日ちょっとやらかした話してもいい？」
相手: 「何何！！教えて！！！」（全力で乗ってくる）
→ 話す → 盛り上がる
```

「笑い」「あるある」「驚き」の3種類のネタを用意すると会話が止まらない。

### この型が効く理由

盛り上がることが大好き。盛り上がりを作ってくれる人を「一緒にいて楽しい人」として本能的に好む。

---

## RFCD（即レス誠実型）の相手に効く返信

### このタイプの読み方

既読スルーは絶対にしない。返信速度は速いが短め。自分から「会いたい」とは言えない。

### 効く返信の型

**「誠実さへの感謝 + こちらから1つ動く」**

```
相手の誠実な返信を受け取ったことを示す → こちらから次の行動を提案する
```

**具体例**:
```
相手: 「うん、そういうことがあったんだよ」（短いが誠実に返している）
型:  「話してくれてよかった。今度もっとゆっくり話せたらいいな。今週どこかいる？」
```

誠実さを受け取り → 関係を進める提案を混ぜる。

### この型が効く理由

誠実に返し続けているのに関係が進まない、という経験を持っているタイプ。こちらから動いてくれる人を、本人が思っている以上に求めている。

---

## RFCP（省エネ即レス型）の相手に効く返信

### このタイプの読み方

来たら速い。でも短い。「わかった」「そうなんだ」で返しがち。質問を返さないことが多い。

### 効く返信の型

**「自分から話題を出す + 選択肢を1つ添える」**

```
こちらが話題を出す → 相手が答えやすい選択肢 or イエスノーで返せる質問を添える
```

**具体例**:
```
あなた: 「今日なんか食べた？ごはん派？ラーメン派？」
相手: 「ごはん」（短いが返ってくる）
```

「どんなものが好き？」より「AとBどっち？」の方が返してくれる確率が上がる。

### この型が効く理由

省エネ型にとって「答えやすい質問」は返せる質問。返しやすい問いを投げることで、短い返信でも会話が続く。

---

## RSED（深海共感型）の相手に効く返信

### このタイプの読み方

返信は遅い。でも来た時の言葉が深い。深夜に突然、核心をついた一言が来ることがある。

### 効く返信の型

**「時間を待つ宣言 + 深い質問を1つだけ置く」**

```
「急がなくていい」というメッセージを先に送る → その上で深い質問を1つ投げる
```

**具体例**:
```
あなた: 「返信いつでもいいから。ちょっと聞いてみたかったんだけど、
        最近で一番嬉しかったことって何？」
```

この2つを同時に送ることで、「プレッシャーがないのに深い話ができる」場が生まれる。

### この型が効く理由

深い話を好むが、プレッシャーがあると閉じる。「焦らなくていい」と「深い問い」を合わせると、このタイプが最も気持ちよく話せる環境になる。

---

## RSEP（気まぐれリアクター型）の相手に効く返信

### このタイプの読み方

気が向いた時は全力でリアクション。そうじゃない時は消える。ムラがある。でも気が合う時の会話は最高に楽しい。

### 効く返信の型

**「来た時に全力で乗る → 来ない時は待つ」の繰り返し**

```
相手が来た時 → 全力で返す（盛り上がらせる）
相手が来ない時 → 追わない（1〜2日後に別の話題を1通だけ）
```

**具体例**:
```
（相手が珍しく盛り上がってきた日）
あなた: [全力で乗る・話を広げる]
→ 翌日: 「昨日楽しかったー！」と1通だけ（引き留めない）
```

盛り上がりを引き延ばそうとせず、「今日は楽しかった」で完結させる。

### この型が効く理由

来た時に楽しい経験を作れる人と、また来たくなる。「また会いたい」は強制では生まれない。

---

## RSCD（慎重観察型）の相手に効く返信

### このタイプの読み方

相手のテンションや絵文字量を観察してから返す。「どう受け取られるか」を先に考える。返信が遅いのは熟考しているから。

### 効く返信の型

**「一貫した誠実さ + 少しずつ自己開示を重ねる」**

```
毎回の返信で、この人は安全で誠実だと確認させる → 少しずつ自分の話を混ぜる
```

**具体例**:
```
（毎回）一貫した文体・温度で返す（急に甘えたり、急に冷たくなったりしない）
（少しずつ）「実は〇〇が好きで」と自己開示を1文ずつ足す
```

「この人は信頼できる」という評価が積み重なると、一気に心が開く。

### この型が効く理由

慎重型は観察して信頼を確認してから動く。一貫性が何より重要。

---

## RSCP（既読温存型）の相手に効く返信

### このタイプの読み方

読んでいるが返さない。悪気はない。返すタイミングを逃し続けている。自然消滅しやすいタイプ。

### 効く返信の型

**「返事が来なくて当然の1通」を週1で送る**

```
返事を求めていない一言 → 義務感ゼロ → たまに返ってきたら全力で受け取る
```

**具体例**:
```
「〇〇見てたらあなたのこと思い出した」（返事を求めない）
「昨日〇〇行ったよ」（報告だけ）
「最近どうしてる？（急がなくていいよ）」
```

返事が来ることを期待しない1通は、プレッシャーがないので一番返ってきやすい。

### この型が効く理由

このタイプにとって最大の敵は「返さないといけない」という義務感。義務感がなければ、気が向いた時に返してくる。

## 第1章のまとめ

| タイプ | 効く返信の核心 |
|---|---|
| AFED | 感情を受け止めて、引き出す質問を返す |
| AFEP | テンポに乗り、話題を広げ続ける |
| AFCD | 中身で誠実に、自分の意見も入れる |
| AFCP | 短く速く、次の話題への橋渡し |
| ASED | 具体的に1つ拾う + 自分の話も少し |
| ASEP | 軽く受け止め、重くしない |
| ASCD | 受け取る + 自己開示 + 相手に戻す |
| ASCP | 程よく短く、余白を持って待つ |
| RFED | こちらから話題を振り、全力の共感を引き出す |
| RFEP | 面白い話を振り、乗ってきたら全力で返す |
| RFCD | 誠実さを受け取り、こちらから1歩動く |
| RFCP | 選択肢付きの答えやすい質問で会話を回す |
| RSED | 急かさずに深い質問を1つだけ置く |
| RSEP | 来た時に全力で盛り上げ、来ない時は追わない |
| RSCD | 一貫した誠実さを積み重ね、少しずつ自己開示 |
| RSCP | 義務感ゼロの1通を週1で送り続ける |
