# 第3章：「これを送ると引かれる」タイプ別NG返信完全ガイド

> 引かれる理由の9割は、悪意じゃなくて「スタイルの不一致」です。
> 知っていれば、防げたのに。

---

## AFED（直感アタッカー型）への NG返信

### NG①：既読放置・そっけない短文

```
❌ 相手:「ねえ、昨日の話続きしたかったんだけど！」
❌ あなた:「うん」（または既読スルー）
```

熱を全力で投げてきた相手に、「うん」で返した瞬間から
相手の心が少しだけ冷える。

次はちょっとだけ送りにくくなる。
そのちょっとが積み重なって、フェードアウトになる。

**代わりに**: 「続きめちゃ聞きたい！話して」と同じ温度で受け止める。

---

### NG②：感情を論理で返す

```
❌ 相手:「怒ってる？なんか変な気がして」
❌ あなた:「怒ってないけど。なんで？理由がわからん」
```

「気がして」と送ってきたのは、安心したかったから。
「理由がわからん」は正しいかもしれないけど、それはいらなかった言葉。

**代わりに**: 「怒ってないよ。心配してくれてたの？ちょっと嬉しかった」

---

## AFEP（ノリよく沼らせ型）への NG返信

### NG①：丁寧すぎる文体で返す

```
❌ 相手:「てかさ〜！昨日の映画なんか笑いすぎて😂😂😂」
❌ あなた:「そうなんですね。どんな内容だったんですか？」
```

この一瞬、相手の中でテンションが半分になった。

笑いすぎた話には「笑いすぎwww それどのシーン！？」でいい。
「そうなんですね」は丁寧じゃなくて、ただ遠い。

**代わりに**: テンポで返す。乗る。笑う。それだけ。

---

### NG②：深い話で引き留めようとする

```
❌「ところで、好きな人いるの？将来どんな恋愛したい？」
```

楽しかった会話がこれで止まる。

このタイプは浅いところから始めたい。
深い話は、相手が自分から開いてきたときに初めて意味を持つ。

**代わりに**: 軽い話を積み重ねて「楽しい」を先に作る。深さは後からついてくる。

---

## AFCD（静かな直球型）への NG返信

### NG①：絵文字・スタンプを過剰に使う

```
❌「わかったー！じゃあね🥰💕✨🎉👋」
```

このタイプの目には「中身のない文章」として映っている。

絵文字が多いほど、言葉の重さが薄くなる。
「ありがとう。また話しかけてね」の方が、ずっと届く。

**代わりに**: 絵文字2個まで。言葉に重さを乗せる。

---

### NG②：遠回しにほのめかす

```
❌「なんか…もしよかったら…いつかまた…一緒に…」
```

伝わっていない。ただそれだけ。

直球型は遠回しな表現を「曖昧なもの」として保留する。
率直に言われた言葉だけを受け取る。

**代わりに**: 「次、一緒に行こう」と言う。それだけで全部伝わる。

---

## AFCP（テンポ職人型）への NG返信

### NG①：長文で話の流れを止める

```
❌ [3段落にわたる自分語り]
```

テンポ職人型にとって、長文は「渋滞」。
先に進めなくなる。

長い内容は「一番大事な一文だけ送る」か、
「短く分けて送る」かで解決する。

**代わりに**: まず一番言いたいことを1行で。続きは相手が返してから。

---

### NG②：返信が3時間以上空く

```
（3時間後）「お待たせ！さっきの話だけど…」
```

「さっきの話」はもう終わっている。相手の中では。

返せない時は「後でゆっくり返す！」の一言が全て。
流れを止めなければ、時間は関係ない。

---

## ASED（じっくり情熱型）への NG返信

### NG①：返信を催促する

```
❌「返信まだ〜？」「見てる？」「既読ついてるよ？」
```

「ちゃんと返したい」から時間がかかっている。
それを催促されるたびに、返すことへのハードルが上がる。

追えば追うほど、遠くなる。

**代わりに**: 待つ。本当に急ぐなら「急がなくていいけど、確認したかっただけ」。

---

### NG②：長文を「w それいいね」で流す

```
❌ 相手: [2段落の思い出話]
❌ あなた:「w それいいね」
```

「この人には本音を送っても大丈夫」と思ったから長文を送った。
「w それいいね」は、その信頼に答えていない。

**代わりに**: 「○○の部分、めちゃくちゃわかった」だけでいい。1つ拾うだけで全然違う。

---

## ASEP（マイペース甘え型）への NG返信

### NG①：返信の波を指摘する

```
❌「返信がバラバラすぎてよくわからない」
❌「急に来るからびっくりする」
```

自分でもわかってる。
それでも指摘されると、送ることへの罪悪感が生まれる。

罪悪感のある相手とのやり取りは、だんだん遠のく。

**代わりに**: 来たら「来てくれた」と受け取る。波があることを責めない。

---

### NG②：軽い話に重く返す

```
❌ 相手:「ねえ聞いてー！なんかあってさ〜」
❌ あなた:「どうしたの。ちゃんと教えて。心配してる」
```

軽くつぶやいただけなのに、心配させてしまった。
そう思うと次から「重い話にしたくないな」が先に来る。

**代わりに**: 「あったの！話して話して！」と軽く受け取る。

---

## ASCD（熟考カウンセラー型）への NG返信

### NG①：相手の開示に何も返さない

相談に乗り続けて、自分のことは一切話さない関係。

「私だけが開いてる」と気づいた時、このタイプは静かに距離を取る。
怒るより先に、閉じる。

**代わりに**: 相談されたら受け止めたうえで、「実は私も〇〇で悩んでる」と返す。

---

### NG②：感情的に詰める

```
❌「なんでそんな返しなの？」「冷たくない？」
```

このタイプは言葉を慎重に選んでいる。
感情で詰められると「安全じゃない」と判断して、さらに言葉が少なくなる。

**代わりに**: 「最近、なんか距離感じてて。気のせいかな」と柔らかく。

---

## ASCP（余白キープ型）への NG返信

### NG①：連投・追いメッセージ

```
❌「ねえ」「見てる？」「返して」「もしかして怒った？」
```

「プレッシャー」しか感じていない。

これは意地悪でも無視でもない。
ただ、追われると逃げたくなる、それだけのこと。

**代わりに**: 1通送ったら返ってくるまで待つ。本当にそれだけでいい。

---

### NG②：まだ距離がある時期に重い好意を伝える

```
❌「好きです。ずっと気になってた」（関係が浅い段階に）
```

このタイプを近づけたいなら、逆に少し引く。
重さが来ると、より遠くへ行く。

**代わりに**: 好意は行動で少しずつ積む。言葉にするのは関係が温まってから。

---

## RFED（反応濃いめ共感型）への NG返信

### NG①：ずっと待ち続ける

「来たら返す」を続けると、ある日相手がふと気づく。
「私ばっかり送ってる」。

気づいた後、来なくなる。

**代わりに**: 週2〜3回、自分からたわいもない一言を。それだけで関係が変わる。

---

### NG②：全力の共感に「だよね」で返す

```
❌ 相手:「それわかりすぎる!!!! 本当にそれ！！！！」
❌ あなた:「だよね」
```

全力で「わかる！」と言ってきた相手に「だよね」は、
温度差が大きすぎて、相手がひとりで盛り上がった形になる。

**代わりに**: 「わかる！しかも〇〇な時とかさ」と、一言加えるだけで全然違う。

---

## RFEP（わちゃわちゃリアクション型）への NG返信

### NG①：盛り上がりにブレーキをかける

```
❌ 相手:「それよりさ！てかさ！聞いて聞いて！」
❌ あなた:「えっとさっきの話まだ終わってないんだけど」
```

盛り上がっている途中で止めると、テンションが急落する。
このタイプは「流れを止めない人」を好む。

**代わりに**: ひとまず乗る。さっきの話は後から「そういえばさっき〜」と笑いに変える。

---

### NG②：自分から動かない

「来てくれるから返す」を続けると、相手が疲れてくる。
そのうち来なくなる。

**代わりに**: 週1回だけ、自分からランダムな一言を送る。それだけで関係の空気が変わる。

---

## RFCD（即レス誠実型）への NG返信

### NG①：誠実さを「当たり前」として受け取る

即レスで、ちゃんとした文章で、毎回返してくれている。
それに対して「ありがとう」も「早いね」も言わない。

やってもやっても届かない、に気づいた時から熱が下がる。

**代わりに**: 「返信速いね、ありがとう」。それだけで相手が続けやすくなる。

---

### NG②：全部向こうからくるのを待つ

即レスし続けているのを「好意のサイン」と解釈して、でも動かない。

このタイプは「来たら返す」が基本行動。
待っていても、関係は前に進まない。

**代わりに**: こちらから「今度〇〇行かない？」と動く。

---

## RFCP（省エネ即レス型）への NG返信

### NG①：重い質問を送る

```
❌「最近どう？何考えてること多い？将来とかについて」
```

「どれに答えればいいの」と思っている。
答えるのが大変で、既読スルーになる。

**代わりに**: 「最近なんか面白いことあった？」と一問に絞る。

---

### NG②：短い返信を責める

```
❌「返信いつも短いよね。あんまり話したくないの？」
```

「誠実に返している」と思っているのに「短い」と言われる。
理解されていない、と感じる。

**代わりに**: 短い返信を補うように、こちらが少し話を広げる。短さは責めない。

---

## RSED（深海共感型）への NG返信

### NG①：既読が続いた後、追いかける

```
❌「ねえ？見てる？」「無視されてる気がする」
```

言葉を選ぶのに時間がかかっている。
追われるたびに「ちゃんと返したいのに」とプレッシャーが増す。

プレッシャーがあると、返すことへのハードルが上がる。

**代わりに**: 「急がなくていいよ」の一言を先に送る。相手が動きやすくなる。

---

### NG②：表面的な会話だけで終わらせる

天気、食事、仕事の話だけが続く関係。

深海型は深く繋がりたい。
浅い会話しかない相手からは、だんだん心が離れていく。

**代わりに**: 「最近どんなこと考えてた？」と、たまに深く入る機会を作る。

---

## RSEP（気まぐれリアクター型）への NG返信

### NG①：返信のムラを責める

```
❌「昨日は全然返してくれないのに、今日はどうしたの」
❌「気分次第なの？」
```

責められると、余計に送りにくくなる。
ムラがあることは、自分でもわかっている。

**代わりに**: 来た時に楽しく返す。来ない時は別の用事をする。追わない。

---

### NG②：盛り上がった日に「毎日これくらい話せたら」と言う

```
❌「毎日このくらい話せたらいいのに」
```

「毎日」というプレッシャーが入ると、
盛り上がること自体を避けるようになる。

**代わりに**: 「今日楽しかった」と今に完結させる。

---

## RSCD（慎重観察型）への NG返信

### NG①：関係が浅い段階で感情を全開にする

```
❌「俺（私）のこと、どう思ってる？」「好きになりそう？」
```

慎重に距離を測っているタイプに、早すぎる感情の開示は
「この人は慎重じゃない」というアラートになる。

引かれるではなく、「保留」になる。
そのまま会わなくなる。

**代わりに**: まず「安全な人」として信頼を積む。それが全て。

---

### NG②：一貫性のない行動を見せる

```
❌ 他の人にも同じような言葉を使っているとわかる行動
```

このタイプはよく観察している。
一貫性のなさを感じた瞬間、静かに距離を置く。

大げさな感情表現より、誠実な日常の言葉の方が信頼される。

---

## RSCP（既読温存型）への NG返信

### NG①：追いメッセージを送る

```
❌「ねえ」「見てる？」「無視？」
```

16タイプ中、追いメッセージが最も逆効果になるタイプ。

1通届くたびに「返しにくい状況」が積み上がる。
5通溜まったら、何と返せばいいかわからなくなる。

**代わりに**: 1通だけ送ったら数日待つ。まったく別の話題を、1通だけ。

---

### NG②：関係の確認を求める

```
❌「最近どう思ってるか教えてほしい」
❌「友達と恋愛、どっちで見てる？」
```

コストの高い質問は逃げられる。
このタイプに答えを求めるより、「答えなくていい距離感」で続ける方が近づける。

**代わりに**: 定期的に「たわいもない一言」を送り続ける。返事がなくてもいい前提で。

---

## まとめ：引かれる返信の共通パターン

| パターン | 特に注意すべきタイプ |
|---|---|
| 追いメッセージ・催促 | ASCP / RSCP / RSCD |
| 重い長文・深い質問（早い段階） | RFCP / AFCP / RSCP |
| 返信ムラを責める | RSEP / ASEP |
| 熱量の差が大きすぎる返信 | AFED / RFED |
| 自分から動かない（待ち続ける） | RFED / RFEP / RFCD |
| 相手の言葉を軽く流す | ASED / RSED |

> 引かれる理由のほとんどは、悪意ではなくスタイルの不一致。
> タイプを知れば、事前に防げる。
