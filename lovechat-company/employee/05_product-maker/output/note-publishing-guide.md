## 事前準備

### noteアカウント作成
1. https://note.com にアクセス
2. 「新規登録」→ メールアドレス or Google/Apple/Twitter で登録
3. ユーザー名設定（例: `lovechat16` または `lovechat_16`）
4. プロフィール設定:
   - 表示名: 「ラブチャット16診断」または「ラブチャット」
   - 自己紹介: 「恋愛LINEの会話タイプを16種類に診断📱 無料診断→ [URL] 」
   - アイコン画像: Canvaで作成したプロフ画像を使用

## Phase 1：無料版記事の公開

**ファイル**: `note-kihon-draft.md`
**公開タイミング**: Xアカウント開設後すぐ

### 手順

1. **noteの投稿画面を開く**
   - 「投稿する」→「テキスト」を選択

2. **タイトルを入力**
   ```
   あなたのLINE、ちゃんと伝わってますか？｜恋愛チャット16タイプ診断 返信設計ガイド（無料）
   ```

3. **本文をコピペ**
   - `note-kihon-draft.md` の `<!-- note に貼る本文ここから -->` 〜 `<!-- ここまで -->` の間のテキストをコピー
   - noteの投稿画面に貼り付け

4. **[診断URL]を実際のURLに差し替え**
   - Ctrl+F（検索）で `[診断URL]` を検索
   - 実際の公開URL（例: `https://kazum.github.io/lovechat16/`）に全て置換

5. **見出しの設定**
   - noteは `##` が自動で見出しになる
   - プレビューで確認してズレてたら手動で設定

6. **ヘッダー画像の設定**
   - Canvaで作ったヘッダー画像（16タイプが並んだデザイン）をアップ
   - なければ後でも可（最初はテキストのみでも公開OK）

7. **ハッシュタグ設定**
   ```
   #恋愛診断 #LINE #恋愛 #LINEのクセ #恋愛あるある #チャット診断 #ラブチャット診断
   ```

8. **公開設定**
   - 「全体公開」を選択
   - 「公開する」をクリック

9. **XでURLを告知**
   - 公開後すぐにX投稿:
   ```
   「LINEのすれ違い、返信速度の問題じゃなくてスタイルの違いだった」
   
   noteに書きました。無料で読めます。
   
   → [noteURL]
   
   #ラブチャット診断 #恋愛診断
   ```

## Phase 2：有料版記事A（4章まとめ）の公開

**ファイル**: note-yuryou-chapter1〜4.md
**想定価格**: 980円
**タイトル**: 「【タイプ別】好きな人のLINEへの正しい返し方大全｜全16タイプ完全解説」

### 手順

1. **投稿画面でnote記事を新規作成**

2. **無料部分を書く（購入前に読める部分）**
   ```
   ## この記事はこんな人向け
   - 好きな人のタイプがわかって、でもどう返せばいいかわからない人
   - 相性はいいのになんかすれ違う、その理由を知りたい人
   - タイプ別に「これを送ると引かれる」を知りたい人
   
   ## 無料で読める前書き
   
   [第1章の最初の1〜2タイプだけ無料で見せる]
   
   ↓ 続きは有料部分
   ```

3. **有料ラインを設定**
   - noteの投稿画面で「有料エリアを設定する」をクリック
   - 分割したい箇所にラインを入れる

4. **残りをコピペ**
   - chapter1〜4の内容を順番に貼り付け
   - 各章の間に区切り線を入れる（`---`）

5. **価格設定**
   - 「販売する」→「980円」に設定

6. **Xで予告→公開の流れ**
   - 公開3日前: 「有料版を準備中です。こんな内容が入ります」（テキスト予告）
   - 公開1日前: 「明日公開します」
   - 公開当日: URL告知投稿

## noteで稼ぐためのコツ

**1. 無料で先に価値を見せる**
無料部分で「こんなに丁寧に書いてある」と感じさせることが購入への最短路。

**2. タイトルにキーワードを入れる**
「既読スルー」「LINE」「タイプ別」などの検索ワードをタイトルに入れる。

**3. Xの告知フォーマット**
```
◯◯について、note書きました。
[記事のポイント1行]
→ [noteURL]
```

**4. 複数回告知する**
同じ記事も告知を変えて3〜5回投稿できる。
「今週の人気記事」「こんな人に読んでほしい」など切り口を変える。
