## 各章の詳細

### 無料パート（試し読み範囲）

- このnoteで何が得られるか（読者のベネフィットを明確に）
- 基礎版の4軸おさらい（A/R・F/S・E/C・D/P）
- 「有料版を読むと、こんな問題が解決する」リスト
  - 「既読スルーされがち」→ 第3章のNG返信を避けるだけで変わる
  - 「相手のLINEが読めない」→ 第1章でタイプ別の解読法がわかる
  - 「どうアプローチすれば？」→ 第4章の相性別ロードマップ

---

### 第1章：16タイプ別「返信の型」（全16タイプ）

各タイプに対して：

**テンプレート**:
```
## ○○タイプの人に送るLINE

### このタイプの読み方
[内側の心理・行動パターン]

### 効く返信の型
[返信のリズム・文体・内容の型]

### 具体例
相手: 「[相手の典型的なLINE]」
あなた: 「[理想の返し方]」

### この型が効く理由
[心理的根拠]
```

**16タイプ分のコンテンツ量目安**: 各300〜500字 × 16 = 4,800〜8,000字

---

### 第2章：状況別スクリプト集

以下の状況ごとに「タイプ別の最適な一言」を提供。

| シチュエーション | 内容 |
|---|---|
| 告白前・脈を確認したい | タイプ別「距離を縮めるLINE」 |
| 付き合いたてで緊張している | タイプ別「自然体で話せるようになる返し方」 |
| マンネリ・返信が減ってきた | タイプ別「温度を取り戻す一手」 |
| ケンカ・既読スルー後の修復 | タイプ別「和解のメッセージ」 |
| 会う約束を自然に作る | タイプ別「誘いのかけ方」 |

各シチュエーション × 4大グループ（AF/AS/RF/RS）= 20〜25パターン

---

### 第3章：NG返信完全ガイド（タイプ別）

「このLINEを送ると引かれる」パターンを16タイプ別に整理。

**テンプレート**:
```
## ○○タイプへの禁じ手

### NG①: [禁止返信のパターン]
なぜNG: [心理的な理由]
代わりに: [代替案]

### NG②: ...
```

**例（RSCP・既読温存型への禁じ手）**:
- NG: 「ねえ、見てる？」「返事ちょうだい」（追いメッセージ）
  - なぜNG: プレッシャーで逃げたくなる。追えば追うほど既読スルーが増える。
  - 代わりに: 1週間に1回、プレッシャーゼロの「たわいもない一言」だけ送る。

---

### 第4章：相性ペア別・関係進展のロードマップ

人気の組み合わせ10ペアに対して、「今の状態 → 次のステップ」を具体的に。

| ペア | 特徴 |
|---|---|
| AFED × RFED | 熱量高い同士。早く動くが、追いすぎると崩れる。 |
| ASCP × RFCD | 余白キープ × 誠実即レス。動く側が固定するとうまくいく。 |
| RSED × AFED | 深い × 積極的。アタッカーが先に誘う、深海が深める形が理想。 |
| RFCP × AFCP | 省エネ × テンポ職人。職人が引っ張れば成立する。 |
| ... | 計10ペア |
