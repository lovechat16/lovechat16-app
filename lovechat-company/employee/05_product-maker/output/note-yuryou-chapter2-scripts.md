# 第2章：状況別スクリプト集

> 「何を送ればいいか」が一番困る瞬間に使う章。
> 状況 × 相手のグループで引いてほしい。

タイプを**4大グループ**に分けて使いやすくしている。
- **AF系**（積極的×即レス）：AFED / AFEP / AFCD / AFCP
- **AS系**（積極的×ゆっくり）：ASED / ASEP / ASCD / ASCP
- **RF系**（受け身×即レス）：RFED / RFEP / RFCD / RFCP
- **RS系**（受け身×ゆっくり）：RSED / RSEP / RSCD / RSCP

---

## シチュエーション1：告白前・脈を確認したい

**目的**: 好意を伝えながら、関係を壊さずに距離を縮める一言。

---

### AF系（積極的×即レス）の相手に

```
「最近、〇〇のこと話しかけてくるの待ってた気がする」
```

**解説**: AF系は行動でリードしてくる。こちらが「待ってた」と伝えることで、「来てよかった」「もっと来ていい」が伝わる。脈を確認する意味でも有効。

相手が「どういう意味？」と聞いてきたら、温度が高い証拠。

---

### AS系（積極的×ゆっくり）の相手に

```
「LINEしてて思ったんだけど、もっと直接話したくなってきた」
```

**解説**: AS系は返信に時間がかかるが、来た時の熱量が高い。「LINEじゃ足りない」は「もっと知りたい」のサインとして伝わる。会う提案への橋渡しにもなる。

---

### RF系（受け身×即レス）の相手に

```
「〇〇さんと話してると、なんか気にならないかな〜って思い始めてる」
```

**解説**: RF系は自分からは動かないが、来た話題には反応する。曖昧な好意を投げることで「私もそう思ってるよ」という反応を引き出しやすい。全力の反応が返ってきたら、温度は高い。

---

### RS系（受け身×ゆっくり）の相手に

```
「返信来るの、なんか楽しみにしてることに気づいた」（送って待つだけ）
```

**解説**: RS系に「好き」を直接伝えると重くなる可能性がある。「気づいた」という自然な言い方で、プレッシャーなく気持ちを届ける。返事が来なくても、相手の中に残る。

---

## シチュエーション2：付き合いたて・緊張している

**目的**: 最初の「よそよそしい」を早く解消する一言。

---

### AF系の相手に

```
「なんか、付き合ってからの方がLINE緊張するね笑」
```

**解説**: AF系は正直に気持ちを伝えることが好き。こちらが先に「緊張してる」と言うことで、「私も同じ」という共感が返ってきやすい。緊張を笑いに変えると関係が一気に縮まる。

---

### AS系の相手に

```
「改めて、よろしくね。なんか最初どう話せばいいか考えちゃう」
```

**解説**: AS系はじっくり考える人。「考えちゃう」は相手の心理と共鳴する言葉。「実は私も」と返ってきたら、そこから深い話が始まる。

---

### RF系の相手に

```
「付き合ったばっかりってなんか不思議な感じするよね。
 普通に話しかけていい？」
```

**解説**: RF系は来たら返す人。「話しかけていい？」と許可を求める形にすることで、安心して「もちろん」と返せる。最初のハードルを下げる一言。

---

### RS系の相手に

```
「なんか普段通りでいいよ、って言いたかっただけ」（送ってあとは待つ）
```

**解説**: RS系は変化に慎重。「普段通りでいい」という言葉でプレッシャーを外す。返事がなくても、「圧がかかってない」という安心感が積み重なる。

---

## シチュエーション3：マンネリ・返信が減ってきた

**目的**: 冷えた温度をさり気なく引き戻す一言。

---

### AF系の相手に

```
「最近なんかバタバタしてた？連絡してなくてごめんね。
 久しぶりに話したくなって」
```

**解説**: AF系は正直な気持ちを評価する。「話したくなった」という素直な気持ちが、相手の積極性を引き出す。謝ることで「マンネリを感じていた」ではなく「寂しかった」として伝わる。

---

### AS系の相手に

```
「最近どう？なんかゆっくり話せてなかった気がして」
```

**解説**: AS系は深い話を好む。「ゆっくり話せてなかった」は「深く話したい」の言い換え。マンネリ感を相手も感じているなら、この一言が「またちゃんと返そう」のきっかけになる。

---

### RF系の相手に

```
「なんか面白いことあった？最近話せてなかったから聞きたくて」
```

**解説**: RF系は来た話題に全力で乗る。「面白いことあった？」は全力の共感が返ってきやすい質問。「なかった〜」でも会話のとっかかりになる。

---

### RS系の相手に

```
「たまに思い出してる（それだけ）」
```

**解説**: RS系にとって長い連絡はコストが高い。「それだけ」と断言することで義務感ゼロで届く。返事が来なくても、この一言で自然消滅を防げる。

---

## シチュエーション4：ケンカ・既読スルー後の修復

**目的**: 壁を作らずに、関係を元に戻す最初の一歩。

---

### AF系の相手に

```
「ごめん。言い方が悪かった。もう少し聞いてから言えばよかった」
```

**解説**: AF系は感情を正直に伝えることを評価する。「自分の何が悪かったか」を具体的に伝えることで「ちゃんと考えてくれた」になる。「ごめん」の後に理由を添えるのが鍵。

---

### AS系の相手に

```
「少し時間が経ったけど、まだ気になってる。
 落ち着いたら話せたらいいな」
```

**解説**: AS系は時間をかけて感情を整理する。「少し時間が経った」と伝えることで「気持ちを落ち着けてから伝えてくれた」になる。急かさない姿勢が和解の鍵。

---

### RF系の相手に

```
「ちゃんと仲直りしたかっただけ。返事なくていいよ」
```

**解説**: RF系は自分からは動けないが、来た話に反応する。「返事なくていいよ」で義務感を外すと、逆に返ってきやすい。短くて潔い謝罪が、RF系には届きやすい。

---

### RS系の相手に

```
「前の件、まだ気にしてたらごめん。
 あなたとは普通に話したい」（返事を期待せず送る）
```

**解説**: RS系は義務感が強い返信を嫌う。「返事を期待していない」雰囲気で送ることがポイント。「普通に話したい」は「これ以上追いたくない」とは違う、前向きな一言として届く。

---

## シチュエーション5：会う約束を自然に作る

**目的**: 「行こう」を引き出す、もしくは自然に提案する一言。

---

### AF系の相手に

```
「今週どっかで時間ある？ちょっと話したいことあって」
```

**解説**: AF系は直接的な提案に応えやすい。「話したいことがある」と前置きすることで「何の話？」と気になる効果も。具体的な提案の前に「会いたい」の理由を置くのが効く。

---

### AS系の相手に

```
「〇〇の話、もっとゆっくり聞きたいな。
 今度直接話せたらいいな」
```

**解説**: AS系は深い会話を好む。「LINEだけじゃ足りない」を匂わせることで「会う理由」が自然にできる。「いいな」と柔らかく留めることで、プレッシャーなく「じゃあいつ？」が返ってきやすい。

---

### RF系の相手に

```
「〇〇行ってみたいんだけど、一緒に行く人いなくて。
 よかったら来ない？」
```

**解説**: RF系は「誘ってもらえる」のを待っている。「一緒に行く人いなくて」という理由をつけることで、相手が「行ってあげる」という形で誘いに乗りやすくなる。

---

### RS系の相手に

```
「今度、もし気が向いたら〇〇行こう。急がなくていいよ」
```

**解説**: RS系には「もし」「急がなくていい」がセット。プレッシャーがゼロの誘いは、逆に「気が向いたら行こうかな」になりやすい。断りやすいからこそ、OKが来た時は本気の意味がある。

---

## 第2章のまとめ

| 状況 | AF系 | AS系 | RF系 | RS系 |
|---|---|---|---|---|
| 告白前 | 「待ってた」 | 「直接話したい」 | 曖昧に好意を投げる | 「楽しみにしてた」と置く |
| 付き合いたて | 緊張を笑いに変える | 「どう話せばいいか」と共感 | 「話しかけていい？」 | 「普段通りでいい」 |
| マンネリ | 「話したくなった」と素直に | 「ゆっくり話せてなかった」 | 「面白いことあった？」 | 「たまに思い出してる」 |
| ケンカ後 | 自分の何が悪かったかを具体的に | 時間を置いてから届ける | 短く潔く・義務感ゼロ | 「返事なしでいい」で置く |
| 会う誘い | 直接的に「時間ある？」 | 「ゆっくり話したい」で理由を作る | 「一緒に行く人いなくて」 | 「もし気が向いたら」 |
