# 第3章：「これを送ると引かれる」タイプ別NG返信完全ガイド

> 恋愛LINEで一番多い失敗は「悪意がないのに距離が開く」こと。
> 相手のタイプを知れば、引かれる返信を事前に防げる。

---

## AFED（直感アタッカー型）への NG返信

### NG①：既読放置・そっけない短文

```
❌ 相手: 「ねえ、昨日の話続きしたかったんだけど！」
❌ あなた: 「うん」（または既読スルー）
```

**なぜNG**: アタッカー型は感情をダイレクトに送ってくる。それを「うん」で返すのは、熱量を真正面から跳ね返すこと。一度冷えると、次は送ってきにくくなる。

**代わりに**: 「続きめちゃ聞きたい！話して」など、同じ熱量で受け止める。

### NG②：感情を論理で返す

```
❌ 相手: 「怒ってる？なんか変な気がして」
❌ あなた: 「怒ってないけど。なんで？理由がわからん」
```

**なぜNG**: 感情を確認してきているのに、論理で返すと「距離感のある人」に映る。感情型は「一緒に感じること」を求めている。

**代わりに**: 「怒ってないよ。むしろ心配してくれてるの？ちょっと嬉しかった」

---

## AFEP（ノリよく沼らせ型）への NG返信

### NG①：重い真面目モードで返す

```
❌ 相手: 「てかさ〜！昨日の映画なんか笑いすぎて😂😂😂」
❌ あなた: 「そうなんですね。どんな内容だったんですか？」
```

**なぜNG**: 丁寧すぎる敬語 or 落ち着いた文体は「温度差があるな」と感じさせる。テンポを止める返信はノリよく沼らせ型にとって一番つまらない体験。

**代わりに**: 「笑いすぎw それどのシーン！？」と同じテンポで乗る。

### NG②：深い話で引き留めようとする

```
❌ 「好きな人いるの？将来どんな恋愛したい？」
```

**なぜNG**: このタイプは浅く広く話したい。深い質問が来ると、かわして逃げる。追いかけるほど遠ざかる構造になっている。

**代わりに**: 軽い話を積み重ねて「楽しい」を先に作る。深い話は向こうが開いてから。

---

## AFCD（静かな直球型）への NG返信

### NG①：絵文字・スタンプ過多の返信

```
❌ 「わかったー！じゃあね🥰💕✨🎉👋」
```

**なぜNG**: 絵文字が少ない直球型にとって、絵文字過多の返信は「中身がない」に見える。表面的な明るさより、言葉の重さを好む。

**代わりに**: 「ありがとう。また話しかけてね」くらいの方が刺さる。

### NG②：遠回しにほのめかす

```
❌ 「なんか…もしよかったら…いつかまた…一緒に…」
```

**なぜNG**: 直球型は率直さを好む。遠回しな表現はただ「伝わらない」だけで終わる。

**代わりに**: 「次、一緒に行こう」とシンプルに言う。直球型はこの率直さを好む。

---

## AFCP（テンポ職人型）への NG返信

### NG①：長い長文で話が止まる

```
❌ [3段落にわたる自分語り]
```

**なぜNG**: テンポを命と思っている。会話のリズムを崩す長文は、次の話題に行けなくなる「渋滞」になる。

**代わりに**: 長い内容は「短く分けて送る」か「一番大事な一文だけ送る」。

### NG②：返信が遅い

```
（3時間後に返信）「お待たせ！さっきの話だけど…」
```

**なぜNG**: テンポ職人型は会話の流れを読む。3時間後の「さっきの話」は、流れが完全に切れている。

**代わりに**: 返せない時は「後でゆっくり返す！」と一言入れる。テンポを止めず、存在を示す。

---

## ASED（じっくり情熱型）への NG返信

### NG①：返信を催促する

```
❌ 「返信まだ〜？」「見てる？」「既読ついてるよ？」
```

**なぜNG**: このタイプは「ちゃんと返したい」から時間がかかる。催促されると「ちゃんと返したいのに」というズレが生じて、送りにくくなる。

**代わりに**: 待つ。本当に急ぐなら「急がなくていいけど、確認したかっただけ」と一言。

### NG②：軽く流す

```
❌ 相手: [2段落の思い出話]
❌ あなた: 「w それいいね」
```

**なぜNG**: 長文で返してきた = 「この人には本音を送っても大丈夫」と判断した証拠。それを軽く流すと「やっぱり届かなかった」になる。

**代わりに**: 1つでも具体的に拾う。「〇〇の部分、めちゃくちゃわかった」だけでいい。

---

## ASEP（マイペース甘え型）への NG返信

### NG①：連絡の波を責める

```
❌ 「返信がバラバラすぎてよくわからない」「急に来るからびっくりする」
```

**なぜNG**: 自分でも「気分で動いてる」とわかっている。そこを責められると、送ることへの罪悪感が生まれて離れる。

**代わりに**: 来た時に「来てくれた」と受け取る。波があることを責めない。

### NG②：重い返信で答える

```
❌ 相手: 「ねえ聞いてー！なんかあってさ〜」
❌ あなた: 「どうしたの。ちゃんと教えて。心配してる」
```

**なぜNG**: 軽くつぶやいたことに重く返されると、「こんなに心配させちゃった」とかえって引く。

**代わりに**: 「あったの！話して話して！」と軽く受け取る。

---

## ASCD（熟考カウンセラー型）への NG返信

### NG①：自分のことを話さない

相手が相談に乗り続け、こちらが何も開示しない関係。

**なぜNG**: 相談を受けながら「自分のことを話してくれない人」と感じ、「私だけが開いてる」と思い始める。

**代わりに**: 相談されたら受け止めたうえで、「実は私も〇〇で悩んでることがある」と自己開示を返す。

### NG②：感情的に詰める

```
❌ 「なんでそんな返しなの？」「冷たくない？」
```

**なぜNG**: このタイプは言葉を慎重に選ぶ。感情的に詰められると防衛モードに入り、さらに言葉が少なくなる。

**代わりに**: 「最近、なんか距離感じてて。気のせいかな」と柔らかく伝える。

---

## ASCP（余白キープ型）への NG返信

### NG①：連投・追いメッセージ

```
❌ 「ねえ」「見てる？」「返して」「もしかして怒った？」
```

**なぜNG**: 余白キープ型にとって連投は「プレッシャー」。プレッシャーを感じると逃げたくなる。これは性格の問題ではなくスタイルの問題。

**代わりに**: 1通送ったら返ってくるまで待つ。返ってこなければ、翌日に別の話題を1通だけ。

### NG②：重い好意を先に出す

```
❌ 「好きです。ずっと気になってた」（まだ距離感のある時期に）
```

**なぜNG**: 余白を好む相手に、いきなり重い好意を投げると一気に引かれる。「追わせる」スタイルの相手には、こちらも少しだけ引いた方が近づける。

**代わりに**: 好意は行動で少しずつ積み上げる。言葉にするのは関係が温まってから。

---

## RFED（反応濃いめ共感型）への NG返信

### NG①：自分から送らない

「来たら返す」だけで関係を維持しようとすること。

**なぜNG**: このタイプは「自分からは送りにくい」と感じている。ずっと相手から送ってもらうのを待っていると、「私ばっかり送ってる」と感じさせる。

**代わりに**: 週2〜3回、自分からたわいもない一言を送る。

### NG②：共感への返しが薄い

```
❌ 相手: 「それわかりすぎる!!!! 本当にそれ！！！！」
❌ あなた: 「だよね」
```

**なぜNG**: 全力で共感してきた相手に「だよね」は温度差がありすぎる。

**代わりに**: 「わかる！しかも〇〇な時とかさ」と一言加えるだけで全然違う。

---

## RFEP（わちゃわちゃリアクション型）への NG返信

### NG①：盛り上がりを止める

```
❌ 相手: 「それよりさ！てかさ！聞いて聞いて！」
❌ あなた: 「えっとさっきの話まだ終わってないんだけど」
```

**なぜNG**: 盛り上がりの途中でブレーキをかけると、テンションが急激に下がる。このタイプは「流れを止めない人」を好む。

**代わりに**: ひとまず乗る。さっきの話は「あ、さっきの話ちょっと待って！」と笑いに変える。

### NG②：自分から動かない

「来てくれるから来たら返す」を続けると、相手が疲れて来なくなる。

**代わりに**: 週に1回は自分からランダムな一言を送る。それだけで関係が変わる。

---

## RFCD（即レス誠実型）への NG返信

### NG①：誠実さを「無難」と取る

```
誠実に返してくれることを当然と思い、感謝もリアクションもしない。
```

**なぜNG**: 即レスで誠実に返しているのに、何の反応もない状態が続くと「やってもやっても伝わらない」になる。

**代わりに**: 「返信速いね、ありがとう」など、誠実さを受け取ったと示す。

### NG②：好意サインを待って動かない

```
相手が即レスし続けているのを「好意がある証拠」と解釈して、でも自分からは動かない。
```

**なぜNG**: 即レス誠実型は「来たら返す」が基本。待ち続けても、こちらから誘わないと関係は進まない。

**代わりに**: こちらから具体的に「今度〇〇行かない？」と動く。

---

## RFCP（省エネ即レス型）への NG返信

### NG①：長文・重い質問

```
❌ 「最近どう？何考えてること多い？将来とかについて」
```

**なぜNG**: 省エネ型は返信コストを低く保ちたい。重い質問は「答えるのが大変」になり、既読放置につながる。

**代わりに**: 「最近なんか面白いことあった？」など軽く短く。

### NG②：短い返信を責める

```
❌ 「返信いつも短いよね。あんまり話したくないの？」
```

**なぜNG**: 省エネ型は「短い返信＝誠実に返している」と思っている。それを責めると「理解されない」と感じる。

**代わりに**: 短い返信を補うように、こちらが少し長く話を広げる。

---

## RSED（深海共感型）への NG返信

### NG①：既読スルー扱いで焦って追う

```
❌ 「ねえ？見てる？」「無視されてる気がする」
```

**なぜNG**: 深海型は言葉を選ぶのに時間がかかる。追われると「ちゃんと返したいのに」と焦って、さらに返しにくくなる。

**代わりに**: 「急がなくていいよ」の一言を先に送っておく。これで相手が動きやすくなる。

### NG②：浅い話だけで終わらせる

```
（会話が続いているが、ずっと天気や食事の話）
```

**なぜNG**: 深海型は深い話を好む。表面的な会話だけだと「この人と深く繋がれないのかも」と感じて徐々に話す意欲がなくなる。

**代わりに**: たまに「最近どんなこと考えてた？」と深く入る機会を作る。

---

## RSEP（気まぐれリアクター型）への NG返信

### NG①：返信ムラを責める

```
❌ 「昨日は全然返してくれないのに、今日はどうしたの」
「気分次第なの？」
```

**なぜNG**: 責められると罪悪感で余計に送りにくくなる。ムラがあることは自覚していることが多い。

**代わりに**: 来た時に楽しく返す。来ない時は別の用事をする。追わない。

### NG②：気が向いた時の盛り上がりを持続させようとする

```
（いい感じに盛り上がった日に）
❌ 「毎日このくらい話せたらいいのに」
```

**なぜNG**: このタイプは波があることが自然。「毎日」というプレッシャーが入ると、盛り上がること自体を避けるようになる。

**代わりに**: 「今日楽しかった」と今に完結させる。

---

## RSCD（慎重観察型）への NG返信

### NG①：ガツガツした態度

```
❌ 「俺（私）のこと、どう思ってる？」「好きになりそう？」
（関係の浅い段階で）
```

**なぜNG**: 慎重に距離を測っているタイプにとって、早すぎる感情の開示は「この人は慎重じゃない」というアラートになる。

**代わりに**: 信頼を積み重ねるフェーズを急がない。まず「安全な人」と思ってもらうことが先。

### NG②：軽薄な印象を与える行動

```
❌ 他の人にも同じような返信をしているとわかる行動
（「◯◯ちゃんにも同じこと言ってたよね」など）
```

**なぜNG**: このタイプは観察している。一貫性がない人・誠実でない人を素早く察知して、距離を置く。

**代わりに**: 誠実に、この人だけに向けた言葉を使う。

---

## RSCP（既読温存型）への NG返信

### NG①：追いメッセージ

```
❌ 「ねえ」「見てる？」「無視？」
（既読スルーが続いた時に連続送信）
```

**なぜNG**: 16タイプ中、追いメッセージが最も逆効果になるタイプ。送られるほど「返しにくい状況」が積み上がる。

**代わりに**: 1通だけ送ったら、数日間は待つ。まったく別の話題を1通だけ。

### NG②：重い好意・関係の確認

```
❌ 「最近どう思ってるか教えてほしい」「友達と恋愛、どっちで見てる？」
```

**なぜNG**: このタイプは「返信コストが低いまま関係を維持したい」という無意識がある。重い質問はコストが高い。コストが高いと逃げる。

**代わりに**: 「ちょっとした報告」「たわいもない一言」を定期的に。返事がなくてもよい前提で。

---

## まとめ：引かれる返信の共通パターン

| パターン | 特にNG なタイプ |
|---|---|
| 追いメッセージ・催促 | ASCP / RSCP / RSCD |
| 重い長文・深い質問（早い段階） | RFCP / AFCP / RSCP |
| 返信ムラを責める | RSEP / ASEP |
| 熱量の差が大きすぎる返信 | AFED / RFED |
| 自分から動かない（待ち続ける） | RFED / RFEP / RFCD |
| 相手の言葉を軽く流す | ASED / RSED |

> 「引かれる理由」のほとんどは、悪意ではなくスタイルの不一致。
> タイプを知れば、事前に回避できる。
