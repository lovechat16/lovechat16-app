## 制作方針

- **尺**: 15〜30秒（視聴完了率70%目標）
- **フォーマット**: テキストスタック型（背景は静止画 or シンプルグラデ）
- **冒頭1秒**: 強烈な「言い当て」でスクロール停止を狙う
- **ナレーション**: テキスト表示 + ボイスオーバー（テキスト読み上げ機能活用）
- **オーバーレイ**: キャラクターアイコン + タイプ名 + 診断URLを最後に表示

## Script 01｜既読温存型（RSCP）
**タイトル:** 恋愛LINEで自然消滅しやすい人の特徴

### 尺: 18秒

```
[冒頭 0-2秒]
テキスト: 「既読したのに返さない…」

[3-6秒]
テキスト: これ、嫌いじゃないよ。
         ただ返すタイミングを逃し続けてるだけ。

[7-12秒]
テキスト: そしてある日思い出した時には、
         もう遅かったりする。

[13-16秒]
テキスト: ＼ラブチャット16 ／
         16タイプ中、最も自然消滅しやすいタイプが
         診断でわかる。

[17-18秒]
テキスト: あなたは何タイプ？→ 無料診断
         [URL]
```

**ハッシュタグ:** `#既読スルー #恋愛あるある #自然消滅 #ラブチャット16`
**狙い:** 既読スルーは普遍的な共感。サビは「自然消滅しやすい」ショック要素

## Script 02｜省エネ即レス型（RFCP）
**タイトル:** 即レスなのに全然盛り上がらない人の正体

### 尺: 22秒

```
[冒頭 0-2秒]
テキスト: 「なんか…即レスなのに会話続かない人いない？」

[3-7秒]
テキスト: 悪気はない。
         ただ「わかった」「そうなんだ」で返してる。

[8-12秒]
テキスト: 返信速度だけが誠実さじゃない…
         ってわかってるのに直せないやつ。

[13-16秒]
テキスト: これ、省エネ即レス型に多い特徴です。

[17-20秒]
テキスト: 自分のタイプ、知ってる？
         ＼ラブチャット16で診断 ／

[21-22秒]
テキスト: [URL]
```

**ハッシュタグ:** `#即レス #恋愛LINE #ラブチャット16 #MBTI好きな人と繋がりたい`
**狙い:** 「これ私じゃん」共感。もしくは「これ相手じゃん」でシェア

## Script 03｜直感アタッカー型（AFED）
**タイトル:** 既読後3分でそわそわする人の話

### 尺: 20秒

```
[冒頭 0-2秒]
テキスト: 「既読ついてから3分でそわそわ」

[3-6秒]
テキスト: するよね？？

[7-11秒]
テキスト: 盛り上がると句点なしで送り続ける。
         「怒ってる？」って聞いてしまう。

[12-16秒]
テキスト: これ、直感アタッカー型の特徴。
         あなたの恋愛LINE、16タイプで診断できます。

[17-20秒]
テキスト: 思ったことは即送信の人、
         ぜひ診断してみて → [URL]

         #ラブチャット16
```

**ハッシュタグ:** `#既読 #恋愛LINE #あるある #ラブチャット16`
**狙い:** 「3分でそわそわ」は言い当てパワー最強クラス。即RT/シェアが期待できる

## Script 04｜マイペース甘え型（ASEP）
**タイトル:** 気が向いた時だけ全力な人の話

```

[3-6秒]
テキスト: ↑ 突然送ってくる人いるじゃん。

         ただ正直に生きてるだけ。

[23-25秒]
```

**狙い:** 「これ惹かれてしまう」の共感。送る側・受ける側どちらにも刺さる

## Script 05｜反応濃いめ共感型（RFED）
**タイトル:** 話すと気持ちいい人の正体

### 尺: 20秒

```
[冒頭 0-2秒]
テキスト: 「なんかこの人と話すと気持ちいい…」

[3-7秒]
テキスト: 「わかる！」「それって〇〇な感じ？」
         「もっと聞かせて」

[8-13秒]
テキスト: これ全部、自然にやってる人がいる。
         共感しながら深堀りするのが得意なタイプ。

[14-17秒]
テキスト: でも自分からは送れない。
         返信が止まると「なにかしたかな」ってなる。

[18-20秒]
テキスト: これ当てはまる人、ラブチャット16で
         タイプ確認してみて → [URL]
```

**ハッシュタグ:** `#共感 #恋愛 #ラブチャット16 #MBTI好きな人と繋がりたい`
**狙い:** 「これ、自分の話だった」認識から診断へ。または「この人のことかも」で相手診断へ

## 制作メモ（全スクリプト共通）

### 動画制作ツール（無料）
- CapCut（テキストアニメーション・BGM追加）
- Canva（静止画 → 動画、タイプカード画像の活用）

### BGM方針
- 著作権フリーのLoFiまたはchill系
- セリフ読み上げの邪魔にならない音量（BGMは30〜40%）

### 投稿チェックリスト
- [ ] 冒頭1秒で「これは私？」と思わせる言葉があるか
- [ ] 最後に診断URLが見えているか
- [ ] キャプションに診断URLを入れているか（TikTokのリンクはキャプションに入れる）
- [ ] ハッシュタグ5〜7個

### 優先制作順
1. Script 01（既読温存型）← 最もバイラルしやすい「ショック」要素あり
2. Script 03（直感アタッカー型）← 共感度が最も高い
3. Script 02（省エネ即レス型）← 相手に送りたくなる
4. Script 04（マイペース甘え型）← 可愛げ要素でシェアされやすい
5. Script 05（反応濃いめ共感型）← 深い共感層に刺さる
