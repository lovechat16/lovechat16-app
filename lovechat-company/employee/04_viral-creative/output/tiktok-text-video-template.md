# TikTokテキスト動画テンプレート
作成: 04_viral-creative
日付: 2026-06-08

---

## 概要

撮影・編集ゼロで作れるTikTok動画の「テキスト動画」形式。
スクリーンショット → テキスト → 音楽で完結。
制作時間: 1本あたり10〜15分。

---

## テンプレート構成（15秒 / 30秒）

### 基本構成

```
[0.0〜2.0秒] フック画面
フォント大・インパクトのある1行

[2.1〜6.0秒] 問いかけ画面
「あなたはこのタイプ？」

[6.1〜12.0秒] タイプ特徴3連打
1行ずつ順番に表示（テキストアニメ）

[12.1〜15.0秒] CTA
「プロフのリンクから診断できます」
```

---

## 具体的なテキスト例

### 動画①「既読温存型（RSCP）」

**フック（0〜2秒）**
```
「既読温存型」って
知ってる？
```

**問いかけ（2〜6秒）**
```
返信書いたのに
送れない人

これです
```

**特徴3連打（6〜12秒）**
```
・返信下書きが3つある
・「明日送ろう」が3日続く
・16タイプ中 最も自然消滅しやすい
```

**CTA（12〜15秒）**
```
プロフのリンクから
全タイプ無料診断

#ラブチャット16
```

---

### 動画②「直感アタッカー型（AFED）」

**フック（0〜2秒）**
```
LINE既読後
3分でそわそわする人

見て
```

**問いかけ（2〜6秒）**
```
「直感アタッカー型」

これ自分じゃんって
なると思う
```

**特徴3連打（6〜12秒）**
```
・盛り上がると句点なしで送り続ける
・返信が来るたびにスマホ確認
・「返信まだかな」を1時間で3回思う
```

**CTA（12〜15秒）**
```
全16タイプ診断
無料 · 3分

プロフのリンクから
#ラブチャット16
```

---

### 動画③「省エネ即レス型（RFCP）」

**フック（0〜2秒）**
```
追わせるのが
天才的に上手い人の特徴
```

**問いかけ（2〜6秒）**
```
「省エネ即レス型」

来たらすぐ返すけど
深追いはしない
```

**特徴3連打（6〜12秒）**
```
・既読無視しないけど追いもしない
・相手が先に「最近どう？」と送ってくる
・自然と追わせる空気を作れる
```

**CTA（12〜15秒）**
```
自分もこのタイプか
診断してみて

#ラブチャット16
```

## TikTok制作手順（Canva利用）

1. **Canvaアプリ** → 「TikTok動画」テンプレートを選択
2. 背景色: `#1a1130`（ダークパープル）
3. テキストを各スライドに配置（Noto Sans JP Bold 推奨）
4. アクセントカラー: `#ff5d8f`（ピンク）
5. BGM: TikTokの「Lofi Chill」系から選択（著作権フリー）
6. 各スライドのアニメーション: 「フェードイン」または「スライド上」
7. 書き出し → TikTokに直接投稿
