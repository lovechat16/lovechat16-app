## 2. プロフィール画像（アイコン）

### サイズ・形式
- **推奨サイズ**: 400×400px（表示は円形にトリミングされる）
- **形式**: PNG（透過なし、背景色あり）

### デザイン方針

**案A（キャラクター型）**
- chars.js のマスコット「アタッキュン」ベースのキャラクターアイコン
- 背景：ピンク〜コーラルのグラデーション
- キャラクターをシンプルに大きく配置
- 右下に小さく「16」の数字を入れる

**案B（ロゴ型）**
- 背景：ダークピンク or ディープパープル
- 中央に「ラブチャット」を縦書きor横書きで大きく
- 下に「16」を強調
- ハートとLINEバブル（吹き出し）アイコンを組み合わせる

**推奨**: 案A（Xのタイムラインでキャラクターの方が視覚的に引き込まれやすい）

### カラーパレット
| 用途 | カラーコード |
|---|---|
| メイン背景 | `#FF6B9D`（コーラルピンク） |
| サブ | `#C77DFF`（ラベンダー） |
| アクセント | `#FF9A3C`（オレンジ） |
| テキスト白 | `#FFFFFF` |

## 3. ヘッダー画像

### サイズ・形式
- **推奨サイズ**: 1500×500px
- **形式**: PNG
- **注意**: 左端・右端はスマホ表示でトリミングされるため、重要な情報は中央700px内に収める

### デザイン方針

**16タイプ一覧型ヘッダー**

```
[レイアウトイメージ]

┌─────────────────────────────────────────┐
│  ラブチャット16診断            [ハート絵文字]  │
│  LINEの返信でわかる、あなたの恋愛タイプ        │
│                                          │
│  AFED AFEP AFCD AFCP                    │
│  ASED ASEP ASCD ASCP                    │
│  RFED RFEP RFCD RFCP                    │
│  RSED RSEP RSCD RSCP                    │
│                         ↑ 小さく・装飾的に  │
└─────────────────────────────────────────┘
```

**テキスト要素（必須）**
- メインコピー: `LINEの返信でわかる、あなたの恋愛タイプ`
- サービス名: `ラブチャット16診断`
- サブコピー: `16タイプ・相性診断・無料`

**ビジュアル要素（任意）**
- 吹き出し/LINEバブル風のアイコン群
- ハートと診断タイプコードを組み合わせたグリッド
- キャラクター（アタッキュン等）を右端に配置

**背景**
- ピンク → パープルの横グラデーション
- または：濃いネイビー背景に明るいピンクのテキスト（コントラスト高め）

## 4. タイプ紹介カード（投稿用画像）

### サイズ・形式
- **推奨サイズ**: 1080×1080px（正方形）
- **形式**: PNG
- 各タイプ16枚分作成する

### 共通レイアウト

```
┌────────────────────────────┐
│                            │
│  【タイプコード】            │ ← 上部、小さめ
│  ○○型                     │ ← タイプ名、大きく
│  キャッチコピー1行           │ ← サブタイトル
│                            │
│ ─────────────────────────  │
│                            │
│ ・あるある①                │
│ ・あるある②                │
│ ・あるある③                │
│                            │
│ ─────────────────────────  │
│                            │
│    #ラブチャット診断         │ ← 右下、小さめ
│    lovechat16.com           │
└────────────────────────────┘
```

### タイプ別カラー設定

| グループ | タイプ | 背景色 | テキスト色 |
|---|---|---|---|
| 積極×即レス×感情 | AFED, AFEP | `#FF4D6D`（情熱レッド） | `#FFFFFF` |
| 積極×即レス×冷静 | AFCD, AFCP | `#4361EE`（クールブルー） | `#FFFFFF` |
| 積極×ゆっくり×感情 | ASED, ASEP | `#FF9A3C`（温かいオレンジ） | `#FFFFFF` |
| 積極×ゆっくり×冷静 | ASCD, ASCP | `#7B2D8B`（ミステリアスパープル） | `#FFFFFF` |
| 受け身×即レス×感情 | RFED, RFEP | `#FF6B9D`（ピンク） | `#FFFFFF` |
| 受け身×即レス×冷静 | RFCD, RFCP | `#2EC4B6`（ミント） | `#FFFFFF` |
| 受け身×ゆっくり×感情 | RSED, RSEP | `#C77DFF`（ラベンダー） | `#FFFFFF` |
| 受け身×ゆっくり×冷静 | RSCD, RSCP | `#52796F`（セージグリーン） | `#FFFFFF` |

### フォント方針
- タイプ名: ゴシック系・太字（Noto Sans JP Bold 等）
- キャッチコピー: 明朝系 or 丸ゴシック（読みやすさ優先）
- あるある: Noto Sans JP Regular
- ハッシュタグ: 細字・小さめ

## 5. 固定ポスト用「全タイプ一覧画像」

### サイズ・形式
- **推奨サイズ**: 1200×675px（横長、Xのプレビューに最適）
- **形式**: PNG

### レイアウト
```
┌──────────────────────────────────────────┐
│  あなたのLINEタイプはどれ？               │
│                                          │
│  積極的    AFED AFEP AFCD AFCP           │
│            ASED ASEP ASCD ASCP           │
│                                          │
│  受け身    RFED RFEP RFCD RFCP           │
│            RSED RSEP RSCD RSCP           │
│                                          │
│  👇 今すぐ診断 → [URL]                  │
│  ラブチャット16診断                       │
└──────────────────────────────────────────┘
```

各タイプコードの下にタイプ名（小文字）を入れると、一覧性が上がる。

## 6. OGP画像（URLシェア時サムネイル）

07_dev-engineer への依頼内容:

- **サイズ**: 1200×630px
- **内容**: 診断結果タイプ名 + キャッチコピー + 「ラブチャット16診断」
- **動的生成**: タイプコードをURLパラメータで受け取り、タイプ別OGP画像を返す
- **静的代替**: 全タイプ共通のOGP画像（上記の一覧画像を流用可）

## 7. 制作優先順位

| 優先度 | 制作物 | 理由 |
|---|---|---|
| 最優先 | プロフィール画像（アイコン） | アカウント開設に必須 |
| 最優先 | ヘッダー画像 | アカウント開設に必須 |
| 高 | タイプ紹介カード 16枚 | 投稿開始に必要 |
| 中 | 固定ポスト用一覧画像 | アカウント設定時に欲しい |
| 低 | OGP動的生成 | 実装コストが高い。後回し可 |

## 8. Canvaでの制作手順（簡易ガイド）

1. Canvaで「Instagramの投稿（正方形）」1080×1080 を選択
2. 背景色をタイプ別カラーに設定
3. テキストブロックを上記レイアウト通りに配置
4. フォントは「Noto Sans JP」推奨（日本語対応・無料）
5. 16枚作ったら「一括ダウンロード」でPNG保存
6. ヘッダーは「プレゼンテーション（16:9）」で作成し、1500×500にリサイズ
