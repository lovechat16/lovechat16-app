## Instagram の特性（X との違い）

| 項目 | X（Twitter） | Instagram |
|---|---|---|
| 主な形式 | テキスト＋画像 | 画像・カルーセル・Reels |
| 拡散経路 | RT・引用 | 保存・シェア・発見タブ |
| ユーザー行動 | 読む・反応 | 保存・スクロール |
| ハッシュタグ | 数個 | 10〜20個 |
| 動画 | 短尺 | Reels（15〜90秒）が強い |

**Instagramで伸びるコンテンツ**:
- 保存したくなるまとめ系カルーセル
- 見て終わるだけでなく「後で見返す」もの
- 共感→「友達に送りたい」の流れ

## カルーセル投稿 コンセプト

### 1. 「返信が遅い人、全員のんびり屋じゃない」

**テーマ**: 返信速度の誤解を解く（拡散・保存系）

#### スライド構成

**Slide 1（カバー）**
```
返信が遅い人、
全員のんびり屋じゃないから。
←スワイプして
```
背景: ダーク系・目を引く

**Slide 2**
```
返信が遅い理由は
タイプによってぜんぜん違う

① 言葉を選びすぎてる型
② 気分が乗るまで待つ型
③ 相手のテンポに合わせてる型
④ そもそも通知を見てない型
```

**Slide 3**
```
①「言葉を選びすぎてる型」

送る前に3回読み返す。
誤解を生みそうな言葉を全部消す。
気づいたら30分経ってる。

悪意ゼロ。むしろ誠実。
```

**Slide 4**
```
②「気分が乗るまで待つ型」

返信する言葉はある。
でも「今じゃない」と思ってる。

気分が乗ったら、急に長文が来る。
そういうタイプ。
```

**Slide 5**
```
③「テンポに合わせる型」

相手の返信ペースを読んで
自分も同じくらいのペースで返す。

これ、気を使ってるから。
遅いのは無視じゃない。
```

**Slide 6**
```
④「通知を見てない型」

スマホを手放してる時間が長い。
見たら即返す。
でも「見る」のが遅い。

意識的に放置してるわけじゃない。
```

**Slide 7（CTA）**
```
自分がどのタイプか
16問で診断できる👇

#ラブチャット診断

プロフのリンクから
```

**キャプション**
```
返信が遅い人を「冷たい」と思ってたら、
実はそれ誤解かもしれない🤍

返信のペースって、性格よりも「スタイル」の話。
スタイルがわかると、解釈が変わる。

あなたのLINEタイプ → プロフURLから無料診断

#恋愛診断 #LINE診断 #恋愛あるある #既読スルー
#恋愛心理 #片思い #LINE #好きな人 #ラブチャット診断
#恋愛相談 #恋愛心理学 #MBTI好きな人と繋がりたい
```

### Reel 1：「返信が来ない理由、4タイプで全部わかる」（30秒）

**フック（0〜3秒）**
```

その理由、4タイプに分かれてる。
```

```
タイプ1：言葉を選びすぎてる

タイプ3：プレッシャーを感じた
タイプ4：スマホを手放してた

（各タイプ1〜2秒のテキストアニメ）
```

**オチ（25〜30秒）**
```

プロフから診断できる
```

**ナレーション例（音あり版）**
```
「既読ついたのに返ってこない。
それ、4つの理由に分かれてるんだけど…
全部、悪意ゼロなんだよね」
```
