## 2. 万バズの型とラブチャット16への接続

### 優先型（ラブチャット16に最も刺さる）

#### 型2：対比リスト型（いいね帯 144K〜270K）
- **転用フォーマット：**
  ```
  [A型]の人の返信
  → 既読してから30分、気持ちを整えてから送る
  
  [B型]の人の返信
  → 通知来た0.5秒後に返してる
  
  どっちが好きか、返信する前に一度考えてみて。
  ```
- **なぜ伸びるか：** タイプ間の差分が「自分はどっち？」を誘発。RTより「保存+診断シェア」を狙う。

#### 型7：造語命名型（いいね帯 27K・表示10.2M）
- **転用フォーマット：**
  ```
  「送信済み症候群」
  
  送ったメッセージを3回読み返して
  「重くなかったかな」って不安になる現象。
  
  これ、診断したら【直感アタッカー型】ってわかった。
  ```
- **なぜ伸びるか：** 名前のなかった感覚に命名→コメント・引用誘発。「これ私やん！」保存。

#### 型3：言語化・再定義型（いいね帯 27K〜78K・保存率高）
- **転用フォーマット：**
  ```
  「既読スルーは冷たい」は間違い。
  
  正確には「受け取ったサインを返せないほど
  言葉を選んでいる」が正しい。
  
  【熟考カウンセラー型】はそれをずっとやってる。
  ```

#### 型1：逆張り断定一行型（いいね帯 67K〜229K）
- **転用フォーマット：**
  ```
  即レスが「媚び」だと思ってる人、
  即レスしてる側は1ミリも考えてないです。
  
  ただの【省エネ即レス型】なので。
  ```

## 4. 02 Result Copywriterへの指示

### 使えそうな言い回し（観測データから）

| 言い回し | 元観測 | ラブチャット16接続 |
|---|---|---|
| 「〇〇、が正しい」 | 型3「誠実に生きれば→胸を張れるが正しい」 | 既存のラブチャット16結果文の「（タイプ名）です」を「（タイプ名）が正しい」に差し替える候補 |
| 「〇〇症候群」「〇〇病」命名 | 型7「27歳症候群」 | 各タイプのキャッチコピーに造語命名を追加する |
| 「〇〇に見える」対比 | 型2「愛情 vs 遊び」 | 「本人の意図」vs「相手からの見え方」の二段落設計 |
| 「全〇〇の人」 | 型8「注意喚起ありがとう全女子」 | 「全即レス民」「全既読温存民」というCTA |

### 強化すべき感情

- **「言語化してくれた」感** → 現在の結果文が少し説明的すぎる。「その気持ちに名前をつける」一行を各タイプの冒頭に入れる
- **「相手に送りたい」感** → 今のキャッチコピーは自分視点が多い。「相手がこのタイプかも？」という読み方もできる一行を追加

### 修正すべきタイプ優先度

1. **RSCP（既読温存型）** → 「最も自然消滅しやすい」という表現が不安煽り気味。「自分のペースを大事にするから。ただし相手には伝わってない」と再定義する
2. **AFED（直感アタッカー型）** → 「追撃」という言葉を「連投」に柔らかく調整
3. **ASCD（熟考カウンセラー型）** → 「相談できる人止まり」の描写が少し自己否定的。「恋愛に切り替えるトリガーが明確」という前向き再定義を追加

### 具体的な作業指示

> **RSCP結果文の冒頭に以下を追加する：**
> 「温め中、の意味がわかる人にだけ刺さる。」

> **AFED結果文のキャッチを以下に変更候補：**
> 現：「思ったら即送信。感情をのせて深く踏み込む、攻めの会話。」
> 案：「温めない人。感情は送信で完成する。」

## 8. CEO向け判断メモ

### 今日の結論

実際の54件の万バズ観測データから、ラブチャット16に最も接続しやすいのは**型2（対比リスト）・型7（造語命名）・型3（言語化）** の3型。どれも現在のキャラクター設計・タイプ名と相性が高い。

### 採用すべき投稿型トップ3

1. **造語命名型（型7）** → 「温め症候群」など感覚に名前をつける。保存率・スレッド参加率が高い。
2. **対比リスト型（型2）** → 「A型 vs B型」の差分が「自分はどっち？」を自然に引き出す。シェア誘発。
3. **参加型選手権（型5）** → 「まずうちから→あなたは？」の構文。初期フォロワーを会話で獲得。

### 最初に出すべきX投稿

**投稿案03（参加型「まずうちから」）**
→ 初投稿として自己開示し、「あなたは？」で会話を作る。診断URLへの誘導も自然。

### 画像化すべき投稿

**投稿案09（16タイプ一覧カード）**
→ URLなし・画像だけでアルゴリズムに強い。1枚で診断への好奇心を作れる。04_viral-creativeに依頼。

### 今は避けるべき投稿

- **URL直貼り単発型**（外部リンク臭・アルゴリズム不利）
- **「〜のコツ5選」形式**（恋愛テク屋になる）
- **男性向け・女性向け分けた投稿**（現時点では全性別対応のトーンを維持）

### CEOに判断してほしいこと

1. **投稿開始のGO/STOP判断** → Xプロフィール（bio・ヘッダー画像）完成後に投稿開始できるか確認
2. **「まずうちから（診断者自身の結果公開）」投稿のOK** → 診断してリアルな結果画像を添付して良いか
3. **16タイプ一覧カード制作の優先度** → 04_viral-creativeに依頼するか否か

### 次に実行すべき順番
