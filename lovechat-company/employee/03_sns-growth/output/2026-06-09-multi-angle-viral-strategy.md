## 設計の核心

**悪い順番：** 診断サイトがある → 宣伝するためにXを使う
**正しい順番：** Xで好きになる → プロフィール見る → 「え、診断サイトまであるの？」

このためにアカウントは「コンテンツクリエイター」として振る舞う。
診断サイトはあくまで「このアカウントが作ったもの」。
逆ではない。

## 「まさかの公式かい笑」を設計するプロフィール

### bioの設計思想

診断サイトの宣伝を前面に出さない。
「このアカウントが面白い人」として書く。
診断URLはbioの最後に、さらっと置く。

```
（推奨bio）

恋愛LINEのあるあるを言語化してます。
「これ、名前あったの」が起きるアカウント。

▶ 16タイプ診断: lovechat16.github.io

#ラブチャット16
```

→ 「言語化してます」が人間っぽい。
→ 診断URLはある。でも押し付けていない。
→ バズった投稿からプロフィール見て「あ、診断サイトもあるじゃん」の流れを作る。

### ヘッダー画像の設計

「診断サービスのバナー」を置かない。
世界観のある1枚にする。

推奨：キャラが数体並んでいる、暗めのトーンの横長画像。
「なんのアカウント？」と思わせる引力が必要。
「恋愛LINE診断！」と書いてある画像は逆効果。

## 「バズ×驚き」のシナリオ設計

### シナリオA（命名型バズ）

1. 「温め症候群」がバズる（5000いいね超え）
2. 「このアカウント誰？」でプロフィールへ
3. 「恋愛LINEを言語化してるアカウント」だとわかる
4. 他の投稿を遡って読む
5. 「16タイプ診断: [URL]」を見つける
6. 「まさか診断サイトまであるの？試してみよ」

### シナリオB（脳内実況バズ）

1. 「直感アタッカー型の脳内・午後11時」がバズる
2. 「これ私すぎる」「これ彼氏」で引用が連鎖
3. プロフィールへ → 「他のタイプの脳内も見たい」
4. 「自分は何タイプ？」で診断へ

### シナリオC（無音投稿バズ）

1. 「返信を待ってる、恋愛で一番孤独な時間」がバズる
2. 引用・コメントで共感の連鎖
3. プロフィールへ → 「このアカウント、言葉が刺さる」
4. 他の投稿を読む → 16タイプ診断を発見

## 「まさかの公式」を成立させる3条件

1. **コンテンツが単体で完結している**
   → 診断サイトを知らなくても楽しめる投稿が8割
   
2. **アカウントに「人格」がある**
   → 「誰かが書いてる」感が出る。中の人が見える。
   
3. **診断は「おまけ発見」として機能する**
   → バズ投稿には基本URLを入れない（週1本だけ）
   → 診断はプロフィールとピン投稿に置いておく
   → 「気になった人だけが辿り着く」設計

## 具体的な初日投稿（まず1本だけ決める）

**推奨：角度②「直感アタッカー型の脳内・午後11時」**

理由：
- URLなし → アルゴリズムに乗りやすい
- 「これ私」「これ彼氏/彼女」で引用が生まれる
- 笑えるので共有されやすい
- 診断に興味がなくても楽しめる

→ この1本でリーチを作り、フォロワーが「他のタイプの脳内も読みたい」で固定読者になる。
