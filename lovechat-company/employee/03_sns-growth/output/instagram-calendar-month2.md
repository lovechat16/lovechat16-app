## Month 2 新作カルーセルテキスト詳細

### 45日目「これをやると静かに距離が開く」

**スライド1（表紙）**
```
これをやると
静かに距離が開く

相手のタイプ別
やってはいけないこと

#ラブチャット16
```

**スライド2**
```
即レス型（Fタイプ）に
→ 「考えてから返す」を繰り返す

3時間後の返信が続くと
「この人、私に興味ないのかな」になる

悪意じゃなくても
テンポは気持ちとして伝わる
```

**スライド3**
```
じっくり型（Sタイプ）に
→ 「なんで返信遅いの？」と催促する

ちゃんと返したいから
時間をかけているのに

急かされるほど
「返しにくい」が積み上がる
```

**スライド4**
```
感情型（Eタイプ）に
→ 論理的に正論を返す

「わかるよ、でも実際は〜」
「それは論理的に考えると〜」

気持ちを受け取ってほしかっただけなのに
論破された気分になる
```

**スライド5**
```
冷静型（Cタイプ）に
→ 感情で詰める

「なんでそんな言い方なの」
「冷たくない？」

このタイプは感情が薄いんじゃなくて
表現方法が違うだけ
```

**スライド6（CTA）**
```
距離が開くのは
悪意じゃなくてスタイルの違い

相手のタイプを知れば
事前に防げる

👉 相手を診断してみる
（プロフリンクから）

#ラブチャット16 #恋愛あるある #LINE
```
