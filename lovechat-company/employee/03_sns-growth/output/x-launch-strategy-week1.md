## Week 1 投稿カレンダー（7投稿）

```

やってみて → [URL]

```

### Day 2｜あるある投稿（共感RT狙い）

**投稿タイプ:** あるある×自己開示誘導
**狙い:** リポスト + 「私もこれ！」コメント誘発

```
「既読ついてるのに返信来ない」

これ、じつは相手側の「クセ」による。

・熟考型：ちゃんと返したくて時間かかってる
・マイペース型：返せる気分になったら返す
・余白型：わざと置いてる
・深海型：感情の温度測ってから返す

→ どのタイプかは会話ログを見ればだいたいわかる。
自分のタイプを先に知る → [URL]

#恋愛あるある #ラブチャット16
```

---

### Day 3｜タイプ紹介（1タイプ詳細）

**投稿タイプ:** 1タイプ深掘り
**狙い:** 「これ私じゃん」で自発的リポスト

```
【直感アタッカー型の恋愛LINEあるある】

・既読ついてから3分でそわそわ
・気持ちを絵文字じゃなくて言葉で伝えたい
・盛り上がると句点なしで送り続ける
・「怒ってる？」って聞いてしまう

→ 気持ちは伝わるが、熱量で引かれることもある。
あなたは何タイプ？ → [URL]

#ラブチャット16 #恋愛診断
```

### Day 4｜相性投稿（組み合わせ紹介）

**投稿タイプ:** タイプ相性（スレッド誘発）
**狙い:** 「相手はどのタイプだろう」で診断への誘導

```
「テンポ職人型」×「反応濃いめ共感型」

この組み合わせ、かなり噛み合う。

テンポ職人は会話をリズムよく回して
共感型はその話に全力で乗っかって温度を足す。

お互いに「話してて楽しい」しか言わない組み合わせ。

→ 自分と相手のタイプを調べる → [URL]

#ラブチャット16 #相性
```

## TikTok / Instagram 連携メモ

リサーチ結果: X→TikTok→Instagram の順番で展開が合理的。

- X: まず診断テキスト+タイプ紹介でバズを狙う（今週から）
- TikTok: 「タイプ診断してみた」動画形式で参入（X開設1ヶ月後）
- Instagram: カルーセル形式で「タイプ別あるある」（アカウント開設後）
