## 方針

Instagramのカルーセル（複数枚スワイプ）は「保存率が高い」「リーチが伸びる」両立コンテンツ。
診断系コンテンツは「自分のタイプ確認のために保存」という動機が生まれやすい。

各カルーセルは5枚構成:
- 1枚目: フック（スワイプさせる）
- 2〜4枚目: 本文（診断内容・タイプ別ポイント）
- 5枚目: CTA（診断へ誘導）

---

## カルーセル #1「あなたのLINE、3つのタイプに当てはまるか確認して」

### 1枚目（フック）
```
あなたのLINE、
どのタイプ？

3つ確認して →
```
*背景: ダークパープル / アクセント: #ff5d8f*

```
直感アタッカー型

「既読後3分でそわそわする」

```

```
省エネ即レス型

「来たらすぐ返すけど深追いはしない」

当てはまったら → RFCP
```

```
20問 · 3分 · 端末内完結
```

## カルーセル #2「LINE返信で "重い" と思われやすい人の特徴4つ」

### 1枚目（フック）
```
LINE返信で
"重い"と思われやすい人、
4つの特徴

あなたはいくつ当てはまる？ →
```

### 2枚目
```
特徴① 
気づいた瞬間に返信する

相手の起きてる時間に即反応。
「四六時中LINEしてる人」と思われがち。

→ FSタイプ：F（直感）が強め
```

### 3枚目
```
特徴②
「?」で終わる質問を連投する

会話を広げようとして質問しすぎ。
返信が義務みたいになってしまう。

→ ARタイプ：A（積極）が強め
```

### 4枚目
```
特徴③
絵文字・スタンプが多め

温かみを出そうとして逆に圧を出す。
感情の「量」より「解像度」を大切に。

→ ECタイプ：E（感情）が強め
```

### 5枚目（CTA）
```
自分のLINEのクセ、
知っておいた方がいい。

16タイプ診断（無料）で
全部わかります。

→ プロフのリンクから
   #ラブチャット16
```

## 制作メモ（ユーザーへ）

- Canva のカルーセルテンプレート（1080×1080px）を推奨
- 配色: `#1a1130` 背景 + `#ff5d8f` アクセント + 白テキスト
- フォント: 太字ゴシック（スマホで読みやすさ優先）
- 1枚目の1行目を「驚かせる言葉」にすることがフックの鉄則
