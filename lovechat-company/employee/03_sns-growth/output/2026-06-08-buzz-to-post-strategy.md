## 2. 万バズ型分析（観測ベース）

### 型1：逆張りワンライナー型

- 元になった観測投稿: 「王子様？知らね、待ってりゃ来るんじゃね」（229K）
- 冒頭フック: 期待値を一文で裏切る
- 刺激している感情: 共感・笑い・「これ私」・反発
- 伸びる理由: 常識の裏切りが一文で完結。同意も反発もRTになる
- ラブチャット16への転用: 恋愛の「常識」を逆張りするフック文
- 注意点: 毒が強すぎると炎上リスク。「かわいい毒」に留める

### 型2：対比リスト可視化型

- 元になった観測投稿: 「わたしがしたい時→ / 妻がしたい時→」（144K）
- 冒頭フック: 「〇〇の時→ / △△の時→」
- 刺激している感情: 共感・笑い・「なんでわかるの」・「相手に送りたい」
- 伸びる理由: 差分の可視化で「あるある」が爆発する
- ラブチャット16への転用: タイプ別行動の対比、「既読スルー派vs即返信派」など
- 注意点: 冒頭ワードの過激さ調整が必要。対比どちらも面白く

### 型3：言語化・再定義型

- 元になった観測投稿: 「誠実に生きれば報われる→胸を張れる、が正しい」（78K）
- 冒頭フック: 既存の言葉への異議申し立て
- 刺激している感情: 「言語化してくれた」・保存欲・引用欲
- 伸びる理由: 「ずっと言いたかったこれ」を代弁する
- ラブチャット16への転用: 恋愛観の再定義、診断結果文の言語化精度向上
- 注意点: 言語化の精度が低いと「浅い」と批判される

### 型4：謎かけ・余白型

- 元になった観測投稿: 「彼女できたよ」息子のメモ（12K、リプ1.2K）
- 冒頭フック: 情報を意図的に省く
- 刺激している感情: 好奇心・不安・切なさ・「続きが気になる」
- 伸びる理由: 「何があったか」想像させる強制エンゲージメント
- ラブチャット16への転用: 「好きな人の〇〇を見てから、何も言えなくなった」フック
- 注意点: 嘘くさくなると即バレ。誠実さのある余白が重要

### 型5：参加型選手権型

- 元になった観測投稿: 「冷蔵庫選手権しよー」（170K）
- 冒頭フック: 「〇〇選手権しよー、まずはおれから」
- 刺激している感情: 参加したい・仲間感・笑い
- 伸びる理由: リプ動機を直接喚起する
- ラブチャット16への転用: 「好きな人への返信あるある言って」系
- 注意点: 初速が弱いと終わる。先行事例の強さが必要

### 型6：会話スクショ型

- 元になった観測投稿: Tinderのやりとりスクショ（142K）
- 冒頭フック: 「〇〇したらこれで神」
- 刺激している感情: 笑い・共感・「あるある」・リアリティ
- 伸びる理由: 証拠＋笑いの組み合わせが強い
- ラブチャット16への転用: 診断結果カード画像、タイプ別「あるあるLINE」フィクション
- 注意点: 実在感を出しすぎると炎上リスク。フィクション前提で

## 3. ラブチャット16用 X投稿案 10本

### 投稿案01（診断誘導型）

- 元になった万バズの型: 言語化・再定義型
- 投稿本文:
  ```
  「好きな人のことを考えすぎ」より
  「好きな人のことを考えたら止まらない、そういうタイプ」の方が正しい気がする。
  
  あなたの恋愛スタイル、16タイプで言語化してみた。
  → [診断リンク]
  ```
- 狙う感情: 「言語化してくれた」・自己診断欲
- 画像/動画案: 名言カード風テキスト画像（背景: ダーク/ピンク）
- CTA: 診断リンク1本のみ
- リスク: 言語化精度が低いと「ふーん」で終わる
- 改善余地: 「タイプ名」を出すと刺さりが増す可能性
- 採用判定: B（修正後採用）

### 投稿案02（タイプ紹介型）

- 元になった万バズの型: 対比リスト可視化型
- 投稿本文:
  ```
  既読スルーする人→「考えすぎてしまうから、まだ返せない」
  既読スルーされる人→「もう気持ち冷めたのかな」
  
  同じ沈黙に、全然ちがう意味がある。
  あなたはどっち側？
  ```
- 狙う感情: 共感・「なんでわかるの」・「相手に送りたい」
- 画像/動画案: 対比テキスト画像（左右2分割）
- CTA: 「あなたはどっち？→診断で確かめて」（リプライ誘発も兼ねる）
- リスク: どちらが「悪い側」に見えると炎上リスク
- 改善余地: どちらも「あるある」に見える温度感の調整
- 採用判定: A（すぐ使える）

### 投稿案03（会話誘発型）

- 元になった万バズの型: 参加型選手権型
- 投稿本文:
  ```
  好きな人から「ね、今何してる？」って来たときの返し方、
  全員言って。まずわたしから。
  
  「え、何？（←なんでもない）」
  ```
- 狙う感情: 参加したい・笑い・仲間感
- 画像/動画案: テキストのみ（シンプルに）
- CTA: リプ参加 → 盛り上がり後に診断URLをリプで追加
- リスク: 初速弱いと死ぬ
- 改善余地: 複数パターン用意して強い冒頭を選ぶ
- 採用判定: B（Xアカウント初速確認後に判断）

### 投稿案04（タイプ紹介型）

- 元になった万バズの型: 逆張りワンライナー型
- 投稿本文:
  ```
  「好きな人にLINEを送りすぎないようにしよう」

  そういう配慮ができる人が、
  一番ひとりで傷ついてたりする。
  ```
- 狙う感情: 共感・「言語化してくれた」・自己肯定
- 画像/動画案: 縦長テキスト画像（夜感のある暗い背景）
- CTA: なし（保存・RT狙いのシェア型）
- リスク: 「恋愛テク屋」感が出ないよう注意
- 改善余地: 後半に診断CTAを1行で添える版も用意
- 採用判定: A（すぐ使える）

### 投稿案05（画像先行型）

- 元になった万バズの型: 会話スクショ型
- 投稿本文:
  ```
  「ラブチャット16診断したら、わたしが出た」
  （診断結果カード画像）
  
  当たりすぎてしんどい。誰か同じタイプいる？
  ```
- 狙う感情: 共感・「わたしも受けたい」・診断欲
- 画像/動画案: 診断結果カード（タイプ名・説明・キャラクター）画像
- CTA: 「診断はリンクから」（画像が主役なのでテキストは最小限）
- リスク: 診断結果カード画像の制作が必要
- 改善余地: 複数タイプで「このタイプの人いる？」系にも展開可能
- 採用判定: B（結果カード画像の制作が前提）

### 投稿案06（診断誘導型）

- 元になった万バズの型: 謎かけ・余白型
- 投稿本文:
  ```
  好きな人のLINEの返し方を見てから、
  なんか言葉が出てこなくなった。
  
  これが何のタイプかは、診断が教えてくれた。
  ```
- 狙う感情: 好奇心・切なさ・「続きが気になる」
- 画像/動画案: テキストのみ（余白重視）
- CTA: 診断リンク1行
- リスク: 謎かけが露骨すぎると「作ってる感」バレる
- 改善余地: 具体的なシチュエーション（「既読2日後に『ね』だけ来た」等）に変える版も
- 採用判定: B（冒頭の具体性を上げれば採用）

### 投稿案07（タイプ紹介型）

- 元になった万バズの型: 対比リスト可視化型
- 投稿本文:
  ```
  ARタイプの人
  「今日も返信考えすぎて30分経ってた」
  
  FSタイプの人
  「え、もう送ったよ？（5秒前）」
  
  どっちに共感した？
  ```
- 狙う感情: 共感・笑い・「どっちも私」・「相手に送りたい」
- 画像/動画案: タイプ対比カード（2択）
- CTA: 「あなたのタイプを確かめる→診断リンク」
- リスク: タイプ名（AR/FS）が知られてないと伝わらない
- 改善余地: タイプ名を使わず行動描写のみに絞った版で初回展開
- 採用判定: B（タイプ名を使わない版で修正後採用）

### 投稿案08（会話誘発型）

- 元になった万バズの型: 逆張りワンライナー型
- 投稿本文:
  ```
  「気を遣いすぎる人は恋愛が下手」
  って言われたことがあった。

  逆だと思う。
  ちゃんと傷つける人の方が、ちゃんと好きなんだよな。
  ```
- 狙う感情: 自己肯定・「言語化してくれた」・共感
- 画像/動画案: テキスト1枚画像
- CTA: なし（感情高まった後にリプで診断URLを添える）
- リスク: 「説教くさい」と感じる人もいる
- 改善余地: 「ちゃんと傷つける人」の表現を柔らかくする
- 採用判定: B

### 投稿案09（診断誘導型）

- 元になった万バズの型: 参加型＋言語化型の複合
- 投稿本文:
  ```
  恋愛LINEのタイプ診断、受けた人が多すぎて怖い。

  「当たりすぎ」「これ元カレじゃん」「彼氏に送った」

  16タイプあるから、あなたのタイプもたぶんある。
  → [診断リンク]
  ```
- 狙う感情: 共感・「わたしも受けたい」・FOMO
- 画像/動画案: 「受けた人の声」風テキスト画像
- CTA: 診断リンク1行
- リスク: 「作り話くさい」と見える場合がある
- 改善余地: Xアカウント開設後、実際の反応を引用する形に変えていく
- 採用判定: C（Xアカウント開設後に再評価）

### 投稿案10（タイプ紹介型）

- 元になった万バズの型: 謎かけ型＋対比

  ③  好きすぎて言葉が出ない

- 狙う感情: 参加欲・自己診断欲・コメント欲

- CTA: リプ誘発型、診断リンクは後続で
- リスク: ④の存在が切なすぎる人を傷つける可能性
- 改善余地: ④を「実は怖くて逃げたい」に変えると毒が和らぐ

## 5. 02 Result Copywriterへの指示

### 使えそうな言い回し（観測データから抽出）

- 「報われなくても胸を張れる、そういう人」（誠実さの言語化、78K投稿より）
- 「王子様を待たない人。自分で取りに行く人」（強さの肯定、229K投稿の構造から）
- 「ツンデレ落ち」の感動変換（144K、ざまぁww投稿から）
- 「返事ができていない」の余白表現（感情の省略による共感増幅）

### 強化すべき感情

- 「言語化してくれた」感覚（保存率直結）
- 「これ私のことだ」の一人称化（共有欲直結）
- 「相手に送りたくなる」他者転写感（拡散直結）

### 修正すべきタイプ

- ARタイプ（考えすぎ系）：「配慮できるほど傷つきやすい」軸で再言語化
- FSタイプ（即レス系）：「反射ではなく感情が先走る人」として再定義
- DPタイプ（自分軸系）：「冷たいんじゃなくて、自分に正直すぎる人」

### 具体的な作業指示

1. 言語化型の言い換えを使った診断結果文の冒頭フレーズを各タイプ1本ずつ提案
2. 「相手に送りたくなる」結果文の条件（他者転写が起きる文体）を定義
3. 上記3タイプの結果文トップ1段落を改稿候補として提出
