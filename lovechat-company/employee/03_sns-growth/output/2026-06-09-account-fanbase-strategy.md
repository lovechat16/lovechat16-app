# @lovechat16 ファン獲得戦略
**作成日：2026-06-09**
**担当：03 SNS Growth**
**方針：「診断サイトの宣伝アカウント」ではなく「フォローしたいアカウント」を作る**

---

## 根本的な問いかけ

「なぜこのアカウントをフォローし続けるのか？」

診断サイトへの誘導だけなら、1回使えば終わり。
ファンになってもらうには「診断と関係なく、このアカウントが好き」という状態を作る必要がある。

**参照軸：なぜ人はアカウントをフォローし続けるか**
1. このアカウントが「自分のことを分かってくれる」と感じる
2. このアカウントを見ると「面白い・スッキリする・気づきがある」
3. このアカウントのネタを「友達に送りたい」と思う
4. このアカウントの次の投稿が気になる（習慣化）

---

## アカウントの"人格設計"

ラブチャット16は**「ちょっと毒があるけど、絶対に傷つけない」コンセプト**。
このトーンをアカウントの人格として一貫させる。

### 声の特徴
- 断言するが、責めない
- 観察者の目線で、当事者を否定しない
- 恋愛テクを教えない。「あなたはこういう人」と言語化するだけ
- たまに自虐。「これ書いてる側も心当たりある」
- ユーモアは「苦笑い系」。笑えるけど笑えない

### 禁止ワード・禁止トーン
- 「モテる方法」「脈あり確認法」→ テク屋
- 「〜な人は要注意」→ 不安煽り
- 「女性は〜」「男性は〜」→ 分断
- 「頑張れ！」→ 自己啓発臭
- 「いいねお願いします」→ 乞食臭

## コンテンツ5本柱（診断と切り離して機能する）

### 柱①：言語化シリーズ ── 「名前のなかった感覚に名前をつける」

**最高優先度。ファン化の核心。**

まだ名前のない「あるある」感覚を、一言で命名する。
命名された側が「これ、ずっと言いたかった」でコメント・引用・保存が爆発する。

```
例：「温め症候群」
例：「既読の海岸線」（波が引くみたいに既読が増えていく感覚）
例：「反射返信」（考える前に返してしまう）
例：「終わらせられない病」（向こうが会話を終わらせるまで待ち続ける）
例：「返信借金」（返したくてたまらないのに返せない状態）
```

**投稿フォーマット：**
```
「（造語）」

○○な感覚の名前。

[2-3行で説明]

診断するとたいてい○○型。
#ラブチャット16
```

→ 診断URLは不要。命名自体が拡散する。

### 柱②：恋愛事故調査 ── 「ストーリーで共感させる」

**エンタメとして見れる。診断への導線は2番目。**

チャット再現 + タイプ分析。
「フィクションリアリズム型」の応用。事実かどうかより「自分に当てはまるか」が大事。

**月2〜3本ペースで連続シリーズとして出す。**
「恋愛事故調査 #〇〇」という番号で「続きが来る」感を作る。

読者が「うちのことかと思った」「これ、現在進行形」でコメントしやすい設計にする。

---

### 柱③：相手LINEの読み方シリーズ ── 「すぐ役立つ」

**保存率最高。「もし相手が○○型だったら」という仮説型コンテンツ。**

```
例：
「相手のLINEが突然短くなった。5つの理由。」
「"わかった"だけで返ってきた時、タイプ別に本当の意味が違う」
「スタンプだけで返ってくる人の2パターン」
```

→ partner.html への自然な誘導になる。
→ 「相手のタイプ診断してみよう」というモチベーションが生まれる。

---

### 柱④：タイプ解説シリーズ ── 「自分を知る、人を知る」

既存の16タイプ紹介投稿の発展形。
ただし**均一フォーマット禁止**。週ごとに切り口を変える。

```
Week1: 「○○型のあるある」（共感型）
Week2: 「○○型 vs ○○型」（対比型）
Week3: 「○○型は実は〇〇だった」（逆張り再定義型）
Week4: 「○○型へ。」（当事者への手紙型）
```

「当事者への手紙型」は特に強い。

```
例：
「省エネ即レス型へ。
あなたは重くない。
ただ、感情を書くのが得意じゃないだけ。
それを「冷たい」と言ってくる人とは合わない。」
```

→ 「泣いた」「保存した」が出やすい形式。

---

### 柱⑤：問いかけシリーズ ── 「コメントが集まる仕組み」

コメント数はアルゴリズムの中で最も重要なシグナル。
「答えたくなる質問」を週1本出す。

```
例：
「相手から既読がついた。返信が来るまで、あなたは何分待てる？」

例：
「好きな人へのLINEで一番緊張するのはどこ？
A）最初のひと言
B）送信ボタンを押す瞬間
C）読まれるまでの時間
D）返ってきた内容を読む瞬間」

例：
「あなたが一番「やってしまった」と思うLINEのクセ、教えて」
```

→ コメントに返信することで「このアカウント、ちゃんと見てくれてる」という信頼を作る。

## ファン化に必要な3つの行動

### 行動①：コメント返信を怠らない
バズっても、コメントに返さないアカウントはファンを作れない。
「この人（アカウント）は会話をする」という実績が信頼になる。

**ルール：**
- 1投稿あたり最低3〜5個のコメントに返信する
- 特に「共感した」「泣いた」「保存した」コメントには必ず返す
- 返信のトーンも「ちょっと毒があるが傷つけない」を守る

### 行動②：他アカウントの投稿に自然にリプライする
自分の投稿だけでなく、関連する投稿に文脈のあるリプライを入れる。

対象：
- MBTI系・診断系アカウントの投稿
- 「既読スルーされた」「返信来ない」系の呟き
- 恋愛相談系アカウント

禁止：
- 「ラブチャット16でもっと詳しく分かります！」→ 宣伝リプ
- 無関係な投稿への空リプ

良い例：
```
「それ、相手が慎重観察型かもしれないですね。
あのタイプ、脈なしじゃなくて言葉を選びすぎてるだけなんです」
```

### 行動③：シリーズの予告をする
「次はこれを出します」という予告が、フォロー動機になる。

```
例：
「明日、"恋愛事故調査 #03" を出します。
今回は「追いメッセ3連射事件」。
関係者の方はメンション禁止で」
```

## 投稿の黄金比率（週あたり）

| 種類 | 本数 | 目的 |
|---|---|---|
| 言語化・造語型（URLなし） | 1本 | バズ・保存・引用 |
| 問いかけ型（URLなし） | 1本 | コメント・会話 |
| 相手/タイプ解説型（URLなし） | 1本 | 保存・信頼蓄積 |
| 診断誘導型（URLあり） | 1本 | 診断流入 |
| 恋愛事故調査（URLあり） | 隔週1本 | エンタメ・信頼 |

**週4本。質重視。毎日投稿はしない。**

毎日投稿すると1本あたりの品質が下がる。
「次の投稿を待ってる」という希少性がファンを作る。

## フォロー動機の設計

### 「このアカウントをフォローするとどうなるか」の答えを作る

現状：「診断サイトの情報が届く」
目標：「恋愛LINEの読み方が変わる。自分のクセが言語化される。」

**この価値をbioに直接書く：**

```
（案A）
恋愛LINEのクセを言語化してます。
「これ、名前あったの」が起きるアカウント。
#ラブチャット16

（案B）
既読スルーも、即レスも、全部タイプの話。
16分類で、LINEの読み方を変えます。
診断 → [URL]
```

| 診断誘導型（診断50件/本） | 5本 | 実ユーザー獲得 |

**最初の1ヶ月で「言語化型バズ投稿」を2本以上当てることが最重要。**

## 中長期：アカウントが「ジャンルの代名詞」になるために

Xで「恋愛LINE」「チャットのクセ」「既読スルー」といった話題が出た時、
@lovechat16 が引用・言及されるようになることがゴール。

そのために：
1. 造語・命名を毎月2〜3個生み出し続ける（命名したものがXに残り続ける）
2. 恋愛事故調査を蓄積してシリーズブランドを作る
3. 「このアカウントを読んでいると恋愛LINEが読めるようになる」という評判を作る

**最終的に目指すのは：**
「恋愛LINEのことなら @lovechat16 を見ればいい」という、ジャンルのポジション。
