## 🐾 動物の擬人化シリーズ（人間社会を動物で言い当てる）

---

**P01「犬型と猫型」**

```
返信が来たとき

犬型の人：「きた！！！」と飛び起きる
猫型の人：「…（既読）」

どちらも好きだけど
一緒にいると話がかみ合わない。
```

---

**P02「犬の返信スタイル」**

```
犬って、呼んだら全速力で来るじゃないですか。

LINEでそれをやってくれる人のことを
「犬みたいな人」と言うのはどうかと思うけど、
正直すごく好きです。
```

---

**P03「猫の既読無視」**

```
猫に「ごはんだよ」と声をかけて
無視された経験がある人は

好きな人に既読スルーされても
そんなに落ち込まなくていいと思う。

猫はただ、今じゃないだけ。
```

---

**P04「犬が毎回全力な理由」**

```
犬が毎回全力で玄関まで来てくれるのは、
「あなたが帰ってきた」という事実を
毎回はじめて喜んでいるから。

昨日も会った。
でも今日も会えた。
それが嬉しい。

その感覚を忘れた大人が多すぎる。
```

---

**P05「ハムスターの回し車」**

```
好きな人のこと
ずっと考えてしまう人間のことを
ハムスターと呼ぼうと思う。

回し車を回し続けて
同じ場所にいる。

でも止められないし、
止める気もない。
```

---

**P06「猫は言葉を省略する」**

```
猫が「にゃ」だけで全部伝えるように、

言葉数が少ない人ほど
一言一言が重いことがある。

「うん」の重さを
ちゃんと受け取れる人でいたい。
```

---

**P07「犬と猫と人間」**

```
犬みたいに全力で好きと言える人と、
猫みたいに言わなくても伝わると思ってる人が
付き合うと

どちらも正しくて
どちらも間違ってる。
```

---

**P08「動物・診断誘導」**

```
「あなたは犬型？猫型？」

って聞かれると
みんなどっちかに当てはまる気がするけど、

LINEの返し方だけで16タイプに分かれることがわかった。

#ラブチャット16
```

## 👨‍👩‍👧 親のLINEシリーズ（誰でも笑える・誰でも共感できる）

---

**P09「親のLINEの改行」**

```
親のLINEには
なぜか1行ごとに改行がある。

「今日は
寒いから
気をつけて。」

なぜなのか誰も聞けていない。
```

---

**P10「父のLINE」**

```
父からのLINEを
ずっと記録している。

「わかった」
「了解」
「うん」
「着いた」

語彙が4つしかない。
でも全部読んでる。
```

---

**P11「母のスタンプ」**

```
母が使うLINEスタンプが
全部ちょっとずれている。

悲しいことを話したら
「最高！！！」のスタンプが来た。

悪気はない。
全力で励ましてる。
そういうことにした。
```

---

**P12「今大丈夫？」**

```
親から「今大丈夫？」とLINEが来ると
なぜか心臓が止まりかける。

「大丈夫じゃなかったら？」という恐怖は
大人になってもなくならない。

大体「ごはん何食べた？」だったけど。
```

---

**P13「既読だけつく父」**

```
父がLINEに既読をつけて
2日返信しないとき、

あれは怒っているんじゃなくて
どう返すか考えてる。

と、信じることにしている。
```

---

**P14「親のLINEが長い日」**

```
普段1行しか送ってこない母が
急に10行のLINEを送ってきた日のことを
今でも覚えている。

内容は「あんたに会いたい」だった。

10行も使う必要はなかったんだけど、
全部読んだ。
```

---

**P15「深夜の親ライン」**

```
深夜1時に母からLINEが来た。

「起きてる？」

起きてると言ったら
「なんか食べた？」だった。

何も心配しなくていいけど
何か心配していた。
```

---

**P16「親の句読点」**

```
普段「OK」しか送ってこない父が
「わかりました。」と句読点つきで返信してきたとき、

何かあったのかと思って
3回読み直した。

何もなかった。
```

## 🎯 集約型言い切り（感情を最短に圧縮する）

---

**P17「待ってる間ずっと」**

```
返信を待っている間じゅう
ずっとあなたのことを考えていた。

送信から既読まで、全部。
```

---

**P18「名前を呼ばれる瞬間」**

```
好きな人に名前を呼ばれると
その日のことを全部覚えている。

声のトーン、天気、自分の服まで。

それだけで十分な日がある。
```

---

**P19「1ミリも損してない」**

```
好きな人のために時間を使った日は
1ミリも損した気がしない。

どれだけ何も返ってこなくても。
```

---

**P20「送信前の1秒」**

```
送信ボタンを押す前の1秒に
すべてが詰まっている。

送った後には何もない。
```

---

## 😅 ツンデレ落ち型（感情→笑いに転換）

---

**P21「12回確認した」**

```
スマホを1時間に12回確認した。
自分で数えてみた。

ざまぁ。
```

---

**P22「覚えてた」**

```
好きな人が1年前に言ってたことを
まだ覚えてた。

どうかしてる。
自分で言う。
```

---

**P23「既読確認の回数」**

```
送った後に3回読み返して
既読がついてないか5回確認して
返信が来たら7回読んだ。

15回。
これが恋愛のコストパフォーマンスです。
```

---

**P24「名前で検索した」**

```
好きな人の名前で検索した。

何も出てこなかった。
当たり前。
でも確認した。

これを「好きすぎ症候群」と呼ぶことにした。
```

---

**P25「返信の下書き」**

```
返信の下書きを8回書き直した。

最終的に送ったのは「了解です！」だった。

8回は何だったのか。
```

## 💫 全員に当てはまる一行型

---

**P26「言わなかった方が多い」**

```
「言ったこと」より
「言わなかったこと」の方が
ずっと多い。
```

---

**P27「音量を下げる」**

```
好きな人の話をするとき
声の音量が少し下がる人間がいる。

大事なことほど
小さい声になる。
```

---

**P28「後で見返す」**

```
誰かに送ってもらった言葉を
後になって読み返すことがある。

送った側はたぶん覚えていない。
受け取った側はずっと覚えている。
```

---

**P29「嫌いじゃない」**

```
「忙しい」は
「嫌いじゃない」と同義のことが多い。

でもそれを確かめる方法が
誰にもわからない。
```
