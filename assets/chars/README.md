# キャラクター画像の置き場所

ここに各タイプのイラストを置くと、診断結果・イントロ・相性カードに表示されます。

## ファイル名ルール（キャラ名ベース）

`<キャラ名>.<拡張子>` で置く。拡張子は `png` → `webp` → `jpg` → `svg` の順で探索。
（アプリは `index.html` の `CHAR[code].name` を見てこの名前で画像を取りに来ます）

## 推奨仕様
- 正方形（例 1024×1024）、背景は白または透過
- キャラ名を変える場合は `index.html`／`partner.html` の `CHAR` の `name` と**ファイル名を一致**させること（不一致だとSVGフォールバックになる）。
- 生成用プロンプトは [../../CHARACTERS.md](../../CHARACTERS.md) を参照
