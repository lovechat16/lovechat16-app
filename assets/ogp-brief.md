# OGPバナー 制作仕様（あなた／デザイナー向け）

X・LINE等でURLを共有したとき表示されるカード画像。**一目で「恋愛メッセージのキャラ診断」と分かること**が最優先。

## 出力
- **サイズ：1200 × 630 px**（OGP標準・1.91:1）
- ファイル：**`assets/ogp.jpg`** に保存（上書き）。これで即反映。
  - PNGで作った場合は `assets/ogp.png` にして「PNGにした」と一言 → meta と server の設定を合わせる。
- 容量目安：〜300KB（JPG 品質85前後）

## セーフエリア
- X/LINEでサムネ表示・トリミングされる前提で、**重要な文字とキャラは中央寄せ**。
- 左右端・上下端ギリギリに主要素を置かない（切れる）。

## デザイン
- 背景：**ダーク紫 → マゼンタのグラデ**（`#1a1130` → `#2b1645`）。アクセント `#ff5d8f`（ピンク）／`#a06bff`（紫）。サイト本体と統一感を。
- 文字：白＋アクセント。**コントラスト高く、サムネでも読める特大タイトル**。

## 必須コピー（日本語・大きく太く）
1. **メインタイトル（特大）**：`あなたは、どの会話キャラ？`
   - 代替：`恋愛メッセージ 16タイプ診断`
2. サブ：`20の質問で “会話タイプ・相性・事故ポイント” がわかる`
3. バッジ：`無料`／`約90秒`
4. `#ラブチャット16`

## キャラ
- 3〜5体を大きめに配置（例：アタッキュン🔥／きょうモフ🥹／ふかメル🌊／わちゃワン🎉／あまぐま🍮）。
- `assets/chars/<キャラ名>.png` に素材あり。

## チェック（一目テスト）
- スマホで縮小表示しても **「恋愛のメッセージ診断」だと3秒で分かるか？**
- タイトルが読めるか？ キャラが可愛く見えるか？ 無料だと分かるか？

## 公開時
- 独自ドメイン取得後、`index.html`／`partner.html` の `og:image`・`twitter:image` を
  **絶対URL**（`https://ドメイン/assets/ogp.jpg`）に置換（X/LINEのスクレイパーは絶対URL推奨）。

---
※相手診断（partner.html）用に別バナーを作ってもよいが、まずは共通1枚でOK。差し替えたい場合は partner.html の `og:image` を別ファイルに変更。
